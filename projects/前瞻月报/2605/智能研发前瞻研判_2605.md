# 智能研发前瞻研判（2026年5月）

## 总体研判

5月纳入研判的事件共10项，覆盖研发范式、智能研发产品、核心支撑能力、支撑底座四个层面，其中重大变革1项、值得跟踪7项、参考借鉴2项。研发范式层出现标志性判断——AI编程将迈入自主Team时代；产品层变化最为密集，**多Agent化与通用化双线并行**；支撑能力层在Agent记忆自沉淀方面取得实质突破；底座层模型编程能力与推理速度均有显著跃升。整体与我行智能研发规划方向一致，后续保持持续推进与关注。

**研发范式**层，Cursor创始人明确提出软件开发将从Tab补全迈入Agent自主Team纪元，企业75%代码已由AI生成、30%的PR端到端自主完成，研发模式将从单一Agent人机协同迈向以Agent Team为核心的自主研发阶段，我行已开展相关研究与建设。**智能研发产品**层，**一**是多智能体协同走向工程化，协作架构趋于成熟（MiniMax Mavis三角色模型、Anthropic异步多智能体协同），指挥中心与可观测性成为刚需（Claude Code Agent View、Qoder 1.0自主工作台），我行SubAgent建设已引入协作架构并在灵犀IDE中规划Agent View能力；**二**是智能研发工具覆盖范围从IDE扩展到全桌面与多端协同（Codex通用电脑管家、TRAE SOLO三端开放），通用Agent能力成为产品竞争新维度，我行将持续跟踪相关演进。**核心支撑能力**层，Claude推出Dreaming记忆机制使Agent任务完成率提升约6倍，Agent自主记忆与知识沉淀能力是智能体持续进化的关键支撑，我行知识工程建设已涵盖相关方向。**支撑底座**层，GPT-5.5首破ProgramBench零通过率，推理算力的Scaling Law正在重塑编程AI的能力边界；模型竞争进入速度与价格双重博弈，推理加速将降低规模化部署门槛，我行将保持持续关注。
