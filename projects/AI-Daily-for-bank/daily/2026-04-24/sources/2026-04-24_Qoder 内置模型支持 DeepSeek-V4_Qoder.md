刚刚！DeepSeek-V4 正式发布，Qoder 第一时间支持。

DeepSeek-V4 是 DeepSeek 新一代模型，包含代码能力更强的 V4-Pro 与更轻量高效的 V4-Flash 两个版本。相比上一代 V3 系列，V4 在代码、长程任务与 Agent 场景上全面升级。

[庆祝] 代码能力：接近闭源 Frontier

V4-Pro 在 Codeforces 基准上拿下 3206 分，在人类选手榜单位列第 23 位，首次让开源模型在算法竞赛上比肩 GPT-5.4、Gemini 3.1 Pro；在更贴近真实场景的 SWE-Verified 上取得 80.6 分，与 Claude Opus 4.6 Max 打成平手；在 Terminal-Bench 2.0 等 Agent 编码评测上，也稳居开源第一梯队。

[庆祝] 长程编码：大型代码库更稳更快

V4 在架构上做了大幅升级，长程任务下的效率与稳定性显著提升。面对跨文件修改、大型代码库整体理解、长会话 Agent 调度等复杂场景，V4 能更好地把握整体脉络，减少上下文丢失和中断等待。

[庆祝] Agent 能力：工具调用更稳，思考不断链

V4 引入了新的工具调用格式，显著降低 tool-call 出错率；同时在多轮工具交互中完整保留推理链路，复杂 Agent 任务执行更连贯。

欢迎大家在 Qoder IDE / CLI / JetBrains 插件中体验 DeepSeek 最新模型。
