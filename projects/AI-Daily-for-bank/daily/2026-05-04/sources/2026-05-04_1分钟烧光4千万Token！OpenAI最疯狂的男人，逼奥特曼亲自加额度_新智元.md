# 1分钟烧光4千万Token！OpenAI最疯狂的男人，逼奥特曼亲自加额度

> 原文链接：https://mp.weixin.qq.com/s/wa5FAKbbQTIczra9T_ziYQ
> 公众号：新智元

新智元报道

编辑：犀牛 好困

【新智元导读】

1分钟烧光4千万token，奥特曼被紧急摇醒！GPT-5.5王牌炸场让开发者赢麻了，但OpenAI背后却藏着1.4万亿美元填不满的恐怖黑洞。

一分钟，4千万token，灰飞烟灭。

「龙虾之父」Peter Steinberger大概没想到，自己会以这种方式上热搜。

今年2月他被Sam Altman亲自招入OpenAI，负责「下一代个人代理」的开发。

短短几个月，他就已经成了OpenAI内部最疯狂的API消费者之一。

就在刚刚，他在X上晒出了一张令人窒息的截图：

OpenAI API速率限制——每分钟4千万token的配额，被他一口气吃干抹净，额度直接归零。

4千万token是什么概念？大约相当于30万页A4纸的文字量。

一个普通开发者可能一个月才能用完的额度，Peter一分钟就烧光了。

截图下面，他只留了一句话：「brb calling @sama」。等一下，我给老板打个电话。

奥特曼秒回复：马上解决！

还不忘调侃一句：这个数值太低了。

这条推文瞬间炸了。

有网友震惊Peter到底在搞什么：「你这是在建什么？第二个互联网吗？」

Peter回应是ClawSweeper——一个维护OpenClaw代码库的机器人。

值得一提的是，ClawSweeper运行在50个基于GPT-5.5的并行Codex上。

GPT-5.5炸场

龙虾之父1分钟烧穿4千万token

Peter之所以能一分钟烧掉4千万token，背后的「罪魁祸首」正是OpenAI最新的王牌武器——GPT-5.5驱动的Codex。

GPT-5.5被OpenAI定义为「最聪明、最直觉化的模型」。

你可以把一个混乱的、多线程的任务直接丢给它，它会自己规划、调用工具、检查结果、处理歧义，然后一路干到底。

这种能力在代码领域的杀伤力尤其恐怖。

Codex搭载GPT-5.5后，直接引爆了开发者社区。

Sam Altman和Greg Brockman罕见地同时在社交平台连推安利，

开发者的反应甚至比官方还夸张。

有人直接说，这可能是这十年AI领域最重大的时刻。

就连Peter自己，也没能逃过Codex 5.5的「真香定律」。

他在推文中坦言，Codex 5.5展现出的「大模型气质」让他惊喜，但同时也带来

了一个棘手的问题：OpenClaw的整个工具链需要紧急重构。

更值得注意的是奥特曼本人的表态。

一位开发者时说，GPT-5.5已经成为他的「日常必需品」，奥特曼回复说：

「那你就准备好迎接人生的蜕变吧」。

从GPT-5.2到5.3到5.4到5.5，OpenAI维持着大约六周一更的恐怖节奏。

按照这个速度，「人生级变革」可能不是遥远的许诺，而是下一个季度的事。

5月5日，OpenAI还将为GPT-5.5举办一场庆祝派对。

最戏剧性的一幕是：Altman在社交媒体上公开表示，正在跟OpenAI打官司的马斯克「想来就可以来」，还补了一句「世界需要更多爱」。

Codex变身超级应用，全面围剿Claude Code

如果说Codex 5.5是OpenAI的攻城锤，那么「全家桶战略」就是它的包围网。

用户Zibo Gao在试用Codex Mac App后兴奋地写道：

「winning SO HARD」——赢麻了。

他顺手列了一个功能愿望清单：原生编辑器、iOS应用、完整浏览器支持、以及OpenClaw深度集成。

Codex团队核心成员Tibo的回复简洁有力：「All of this and more is coming」——这些全都在路上，还有更多惊喜。

OpenAI正在将Codex从一个命令行工具，扩展为覆盖Mac、iOS、浏览器、IDE扩展的全平台产品矩阵。

与此同时，OpenClaw 5.2也在5月2日完成了重大升级。

默认集成xAI最新的Grok 4.3模型，彻底重写插件架构，根治了长期困扰开发者的npm依赖冲突问题，并针对Discord、Slack、Telegram等多平台进行了深度调优。

这套组合拳的指向非常明确：抢夺AI编程工具的生态制高点，把Anthropic的Claude挤到角落里。

看到这里，你大概以为OpenAI的确是赢麻了。

8520亿估值底下

藏着一个填不满的窟窿

然后，冰水从头浇下。

4月底，WSJ记者Berber Jin的深度调查捅破了窗户纸。

OpenAI没有完成自己的营收目标。

2026年初多个月度销售目标未达标。2025年底冲刺的10亿ChatGPT周活跃用户目标，也没实现（年底实际公布的数字是8亿）。

更致命的是OpenAI已经签下了花不完的钱。

Stargate、甲骨文、AWS、微软Azure加在一起，合同总额超过1.4万亿美元。这个数字，比西班牙和澳大利亚的GDP都大！

CFO Sarah Friar的工作就是搞清楚钱从哪里来。

但问题是，这笔账根本算不过来。

而且事情不止于「分歧」。

据The Information报道，Friar从2025年8月起就不再向Altman汇报，改为向应用业务负责人Fidji Simo汇报。一家大公司的CFO不向CEO汇报，这在硅谷几乎闻所未闻。

更蹊跷的是，Friar随后被排除在了投资者会议之外。甚至，Simo本人也宣布因健康原因短暂休假。

消息出来当天，软银东京暴跌10%，Oracle盘前跌超7%，CoreWeave和AMD被连带拖下水。

Morningstar预测认为，OpenAI的IPO窗口将不得不从2026年Q4推迟到2027年中后期。

Anthropic「趁乱抢食」

更致命的是，竞争对手已经完成了超越。

根据Counterpoint Research Q1 2026数据，全球LLM收入份额，Anthropic为31.4%，OpenAI为29%。OpenAI在这个核心指标上第一次被反超。

ARR方面，Anthropic是300亿美元，OpenAI是240-250亿。15个月前Anthropic还只有10亿，翻了30倍。

代码生成市场方面，Anthropic占42%-54%，OpenAI仅21%。

Ramp数据显示，3月新购AI服务的企业中，65%选了Anthropic，只有32%选OpenAI。

不仅如此，Anthropic月均每用户收入16.2美元，OpenAI只有2.2美元，8倍的差距。

2026年已经高达9亿的周活用户听着吓人，但超过95%是免费的，每一个都在烧钱。

OpenAI在一份泄露的股东信中，把Anthropic列为最大竞争威胁。但信发出去的时候，反超已经是既成事实。

马斯克法庭补刀

甚至，奥特曼都没法专心应对自家的商业危机。

同一周，马斯克起诉OpenAI案在加州奥克兰联邦法院开庭，索赔金额可能高达1800亿美元。

庭审当天，奥特曼本该出席AWS在旧金山的合作活动，却不得不通过录制视频远程亮相。

我的日程，最近实在是身不由己。

法律专家普遍认为马斯克胜诉几率不大。但这不重要。

过去两年里，马斯克不遗余力地将奥特曼塑造为「Scam Altman」，AI时代靠忽悠起家的江湖骗子。

这场公关战造成的认知伤害，比法律判决本身更深远。

产品越炸，烧钱越猛

所以现在的局面是，Codex 5.5引爆开发者圈，GPT-5.5跑分碾压一切，超级应用帝国初具雏形。

但另一面，营收没达标，CFO被架空，Anthropic在收入份额上完成反超，马斯克在法庭上等着补最后一刀。

OpenAI的产品狂热和商业现实之间，裂缝正在扩大。而且产品越好，裂缝越大。

按目前的烧钱速度，OpenAI 2026年预计亏损140亿美元，2027年现金消耗可能飙到570亿，盈利最快要等到2029。

Anthropic那边，训练成本只有OpenAI的四分之一，可能2027年就跑通现金流。

产品越炸，烧钱越猛。用户越多，亏损越大。

奥特曼的神话，远没到盖棺定论的时候。
