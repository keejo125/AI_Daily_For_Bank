# 刚刚，马斯克官宣xAI解散，22万张GPU算力租给Anthropic

> 原文链接：https://mp.weixin.qq.com/s/NsQ1siqUVWZ04bIuB_m9Jg
> 公众号：机器之心

机器之心编辑部

就在刚刚，马斯克正式对外确认，xAI 将被解散。

xAI 及其旗下的大语言模型 Grok（以及社交平台 X 的相关业务）将被整合进 SpaceX 的一个全新子部门，统一命名为 SpaceXAI。

这一解散动作的伏笔其实在 2026 年 2 月初就已埋下。当时 SpaceX 以全资收购的方式收购了 xAI，合并后的实体估值高达 1.25 万亿美元。伴随并购与重组，xAI

内部经历了剧烈的震荡

。截至 2026 年 3 月底，xAI 最初的 11 位联合创始人已全部离职。

马斯克一直倾向于将他名下的企业整合到一个控股框架内。分析指出，将 xAI 并入 SpaceX 有助于他集中管理火箭、卫星、AI 基础设施和数据平台，并将 AI 相关的监管与诉讼风险转移到 SpaceX 更庞大的公司治理边界内。同时，这与 SpaceX 计划在轨道上建立太阳能数据中心的宏大愿景相吻合。

同时，马斯克的 SpaceX 与 Anthropic 宣布达成了重磅协议。

全球最大、最新的超级人工智能集群 Colossus 1 的所有算力，超过 22 万块英伟达 GPU 现在将要用于提升 Claude 的代码生成了。

作为协议的一部分，Anthropic 将获得超过 300 兆瓦的计算能力，该公司还「表示有兴趣」与 SpaceX 合作开发未来的太空算力体系。Anthropic 表示，该协议将直接提升其付费用户 Claude Pro 和 Claude Max 的计算能力。

这份合作对于双方都是重大利好。一方面，它为即将上市的 SpaceX 带来了一位重量级客户，帮助其向投资者推销其 AI 雄心；另一方面，它也帮助 Anthropic 缓解了在对其 AI 编码工具 Claude Code 等产品需求激增后产能受限的问题。过去一段时间，因算力紧张，Claude Code 被设置了各种用量限制，开发者社区颇为不满。

在获得 SpaceXAI 的计算能力后，Claude 的使用限额随之提高，包括 Claude Code 和 Claude API。

Anthropic 宣布，以下三项变更，均于今日生效：

第一，针对 Pro、Max、Team 以及按席位付费的 Enterprise 套餐，Anthropic 将 Claude Code 每五小时的速率限制提高一倍。

第二，移除了 Pro 和 Max 账户在 Claude Code 上的高峰时段速率限制削减。

如果把这则消息和 Anthropic 之前的公告放到一起来看，可以发现他们在算力扩张上的决心。通过这些多云、多供应商的巨额绑定，他们把未来几年顶级的 AI 训练 / 推理算力全锁定了，把 Anthropic 的算力未来给「安排明白了」。

作为该协议的一部分，Anthropic 还表示有意与 SpaceXAI 合作开发数吉瓦级的轨道人工智能算力。

SpaceX 表示，训练和运行下一代这些系统所需的算力，正在超出在关键时间窗口内地面电力、土地和冷却所能提供的极限。SpaceX 是唯一一家具备足够发射频率、入轨经济性和星座运营经验的组织，能够将轨道计算作为一个近期的工程项目而非研究概念来推进。如果工程上的挑战能够被克服，基于太空的计算将提供近乎无限的可持续能源，同时对地球的影响更小。

此前，xAI 一直在加紧建设其 AI 算力基础设施，力图和 OpenAI、谷歌等巨头展开竞争。据报道在建造和运营 Colossus 时，xAI 及其子公司 MZX Tech, LLC 甚至安装并运营了数十台天然气燃烧涡轮机为该设施供电，并声称这样做不需要联邦许可证，因为它们只是临时使用。

可见在 AI 技术的研发上，马斯克是下了血本。不知在今天与 Anthropic 的协议确定之后，xAI 团队将何去何从。

马斯克表示，SpaceX 已将其人工智能训练工作转移到 Colossus 2 上，并且他将向其他致力于造福人类的 AI 公司提供算力，就像 SpaceX 以「公平的条款和价格」为竞争对手发射卫星一样。
