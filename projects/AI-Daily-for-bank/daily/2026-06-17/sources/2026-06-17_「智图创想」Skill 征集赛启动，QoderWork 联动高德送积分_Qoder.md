高德地图能力，Skill 还能这么玩

为了给大家一些灵感，这里展示两个已上线的高德官方 Skill，看看地图能力和 AI Agent 结合后能做什么。

高德地图综合服务是工具型 Skill，封装了高德 Web API 的 POI 搜索、路径规划、周边搜索和热力图可视化能力。用户在 QoderWork 里说：帮我查杭州西湖周边 3 公里的聚餐类餐厅，按评分和距离排序导出明细表。QoderWork 自主调用 POI 搜索接口，筛选、排序、汇总，交付一张包含店名、评分、地址和距离的结构化表格。从需求到结果，全部在 QoderWork 内闭环。适合运营分析、选址调研、密度对比等需要"拿到数据再做判断"的任务。

高德个人地图是应用型 Skill，最终产出不是数据表格，而是一张可以在高德地图 App 中直接打开的个人专属地图小程序二维码。比如你告诉 QoderWork：帮我规划周六从城西银泰出发，先去一家安静的咖啡店，再找一家评分高的日料午餐，下午去有绿植的公园散步。QoderWork 完成 POI 筛选、路线规划和天气查询，生成小程序二维码，发到群里每个人点开高德 App 就能看到完整路线。适合出行规划、朋友聚会、出差接待等需要"把结果直接发出去"的任务。

一个是 API 工具箱，一个是成品生成器。同样的地图能力，不同的封装思路，就是不同的 Skill。你的创意，可以是下一个。

还有更多高德官方 Skill 可参考
