# Meta收购华农校友机器人AI公司，团队并入超级智能实验室

> 原文链接：https://mp.weixin.qq.com/s/jLwHuaDEZ2wro4oquulyww
> 公众号：量子位

闻乐 发自 凹非寺

量子位 | 公众号 QbitAI

小扎又出手了，这次瞄准的是

人形机器人

。

Meta正式完成对机器人AI初创公司

Assured Robot Intelligence（简称 ARI）

的收购。

这家公司专注于机器人智能底层技术，由

华南农业大学、中山大学校友王晓龙

联合创办。

按照规划，ARI团队将整体并入

Meta超级智能实验室

，与Meta旗下Robotics Studio团队深度合作，共同推进下一代人形机器人技术落地。

消息官宣后，谢赛宁第一时间表达了祝贺。

Meta收购ARI，落子人形机器人

不难看出，Meta的这起收购意图非常明确——

补齐自身在机器人AI上的短板，提前卡位人形机器人这一未来智能新赛道。

ARI的核心竞争力，就在于

机器人AI基础模型

研发。

团队主攻复杂动态环境下的机器人感知能力，让机器人能够像人一样理解场景、预测行为、主动适应人类互动，这也是人形机器人真正走向日常实用绕不开的关键难点。

而Meta在2025年专门成立的

Robotics Studio

，本身就聚焦人形机器人底层控制与硬件技术，两者刚好高度互补。

△

图源：Bloomberg

加入Meta后，ARI的核心任务将聚焦

机器人场景专用AI模型优化

，把自家成熟算法能力与Meta的算力、工程资源结合。

借此补上Meta在机器人底层智能、运动控制、自主学习模型上的技术缺口，加快Meta人形机器人板块的研发进度。

不过值得注意的是，就在此次收购官宣前两天，Meta刚刚大幅上调了2026年资本支出预算，将年度支出规模拉高至1250 亿—1450亿美元，钱主要砸向AI数据中心和高端硬件组件。

足以看出，Meta已经悄悄调转战略重心，从过去主打元宇宙，全面转向AI+机器人实体智能赛道。

华农校友领衔

此次收购消息的的焦点，ARI公司，由王晓龙和Lerrel Pinto联手创办。

王晓龙本科毕业于

华南农业大学

，后续在中山大学取得硕士学位、在卡内基梅隆大学（CMU）获得机器人学博士学位，现在担任加州大学圣地亚哥分校（UCSD）副教授。

他的研究方向聚焦

基于视频与物理交互数据的表征学习

。

简单来说，就是让机器人靠看视频、自己动手试错，慢慢看懂现实世界，自主学会各种操作技能。

在创办ARI之前，王晓龙曾担任

NVIDIA

研究员，参与构建机器人领域基础大模型。

ARI的另一位联合创始人是

Lerrel Pinto

，同样拥有卡内基梅隆大学博士学位，是大规模机器人学习领域的顶尖学者。

△

图为Lerrel Pinto和Yoshua Bengio

他的研究覆盖数据与模型、感知表征学习、动作建模、强化学习等方向，长期致力于开发低成本、可开源的机器人平台，让先进技术更易普及。
