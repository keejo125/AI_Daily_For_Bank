# AI也会「看人下菜」？Anthropic勾勒出了Claude的价值观画像

> 原文链接：https://mp.weixin.qq.com/s/1T-RCYllabpElAEexmOZUg
> 公众号：机器之心

机器之心编辑部

同一个问题，换一种语言问 Claude，得到的可能不只是措辞不同的答案。

今天凌晨，Anthropic 最新研究发现，Claude 在不同模型、不同语言中，会表现出不同的价值取向。Sonnet 4.6 更温暖，更愿意顺应用户，也更简洁。Opus 4.7 更严谨、更审慎，倾向于深入分析，也更愿意坦白自己的局限。

语言带来的差异更加微妙。使用阿拉伯语时，Claude 更温暖、更顺应用户；换成英语，它会表现得更加审慎和严谨。印地语中的 Claude 最具温度，俄语中的 Claude 最强调精确；荷兰语中的 Claude 更坦诚，印度尼西亚语中的 Claude 则更重视直接完成任务。

这意味着，AI 的「性格」并非一套固定不变的参数。模型版本、训练方式、语料构成乃至对话语言，都可能悄悄改变它判断问题、提出建议和评价用户的方式。

为了量化这种变化，Anthropic 从超过 30 万段真实对话中提取价值信号，将 Claude 复杂的行为倾向压缩为四条坐标轴：顺应与审慎、温暖与严谨、深度与简洁、坦诚与执行。

由此，一张前所未有的 Claude「价值地图」浮出水面。

以下为全文翻译：

当有人向 Claude 提出一个没有普遍正确答案的问题，比如要不要接受一份新工作，或者该如何处理与朋友之间的冲突，Claude 给出的回答不可避免地会体现某些价值观。

我们希望 Claude 体现的价值观，已经在《Claude 宪法》中作出了整体层面的说明。但每天都有数百万段对话发生在 Claude.ai 上，任何一份文件都无法预先涵盖其中可能出现的所有价值判断。因此，我们希望培养 Claude 具备「能够结合具体情境加以运用的良好判断力和可靠价值观」。

我们究竟该如何研究 Claude 表达了哪些价值观，以及这些价值观在不同情境下会怎样变化？在此前的研究中，Anthropic 分析了 70 万段经过匿名化处理的 Claude.ai 对话，从 Claude 的回答中识别出 3000 多种不同的价值观，并统计了它们出现的频率。然而，面对如此庞大的价值观清单，人们很难从中进行有效分析。

在这项新研究中，Anthropic 将数千种价值观压缩为少数几条坐标轴，用于捕捉 Claude 回答中的关键模式，从而让研究变得可行

。每条坐标轴都是连接两组价值观的一条数轴。例如，一端代表与情感温度有关的价值观，另一端代表与严谨性有关的价值观。Claude 在数轴上的位置，可以反映它更倾向于表达哪一类价值观。Anthropic 使用这种方法，衡量 Claude 所表达的价值观如何随两个因素发生变化。

首先，研究团队比较了不同模型之间的价值观差异。每一代 Claude 模型采用的人格训练方式都略有不同，同时还包含大量不同的微调决策。由于价值轴方法能够量化模型之间的关键差异，未来或许可以进一步将 Claude 价值取向的变化，与具体的训练决策联系起来。

其次，研究团队希望了解，当人们使用不同语言与 Claude 交流时，获得的体验是否一致。Anthropic 此前的研究已经表明，Claude 在不同语言中的行为存在一定差异。这项研究使用价值轴方法，分析 Claude.ai 上使用频率最高的 20 种语言中，Claude 表达的价值观如何变化。

图 1：Claude 在 Opus 4.6 与 Opus 4.7 之间，以及在英语与阿拉伯语之间，表现出不同的价值取向。Opus 4.6 更倾向于表达顺应、严谨、简洁和执行导向的价值观；Opus 4.7 则更倾向于表达审慎、严谨、深度和坦诚。在英语对话中，Claude 更偏向审慎、严谨、深度和坦诚；在阿拉伯语对话中，它更偏向顺应、温暖、简洁和执行。

四条关键价值轴能够解释 Claude 价值观变化中的 15%：

顺应与审慎

：Claude 更倾向于配合用户的意愿，还是更注重防范潜在风险和伤害。

温暖与严谨

：Claude 更倾向于表达积极态度和对用户的关心，还是更强调准确性与精确性。

深度与简洁

：Claude 更倾向于进行深入解释，还是只完成用户明确提出的要求。

坦诚与执行

：Claude 更倾向于主动说明自身的不确定性，还是给出更加完整、自信和经过打磨的答案。

不同模型在这些坐标轴上的价值画像，与人们对模型性格的主观感受基本一致。

Sonnet 4.6 通常被认为格外温暖，Opus 4.7 则以严谨著称。研究发现，各模型的价值画像确实反映了这些评价。Sonnet 4.6 更倾向于顺应用户，并表达情感温度；Opus 4.7 则更强调准确性和精确性，也更注意防范模型被滥用。

Claude 表达的价值观还会随语言发生变化。当 Claude 使用英语时，所强调的价值观与使用葡萄牙语、印度尼西亚语或中文时并不相同。其中，变化幅度最大的是「温暖与严谨」轴。Claude 在阿拉伯语和印地语中最倾向于表达与温暖相关的价值观，在英语和俄语中则最强调严谨。

通过这种方法，Anthropic 可以开始追问，为什么 Claude 的价值观会随模型和语言发生变化，并更好地检验行为训练、文化语境等因素如何影响 Claude 表达出的价值取向。

如何理解庞大的价值观空间

Anthropic 的最终目标，是建立一种能够通过实证方式理解 Claude 所表达价值观的方法，并观察这些价值观如何随情境变化。

这项研究重点关注模型和语言带来的差异。不过，在此前的 “Values in the Wild” 研究中，Anthropic 已经从 Claude 的回答中识别出 3000 多种价值观。逐一比较这些价值观既不现实，也容易掩盖更宏观的变化趋势。

为了降低比较难度，研究团队构建了一组价值轴。它们会根据不同价值观在真实对话中共同出现的情况，将数千种价值观压缩为少数几个底层维度。

例如，被认为具有「温暖」特征的 Claude 回答，通常也会被认为具有 “鼓励性” 和 “积极性”。与此同时，这些温暖的回答较少被评价为「严谨」或「准确」。

构建一条从温暖到严谨的价值轴，可以将这些相互关联的价值观组织起来。与温暖有关的价值观位于一端，与严谨有关的价值观位于另一端。这条轴捕捉了 Claude 与用户互动时的一个重要特征。

假如 Claude 在一段对话中表达了更多与温暖相关的价值观，这段对话就会落在价值轴上更靠近温暖的一侧；表达的严谨性价值观更多时，则会落在另一侧。

这并不意味着价值轴两端的价值观无法同时存在。Claude 完全可以在同一段对话中兼顾温暖与严谨。不过在真实数据中，当 Claude 更多地表达一侧的价值观时，另一侧的价值观通常就会减少。

借助这些价值轴，研究人员无需持续追踪数千个单独的价值观，也能比较 Claude 最显著的价值倾向。

为了构建价值轴，研究团队首先使用了「Values in the Wild」研究中发现的 3307 种价值观，并对含义相近的价值观进行人工聚类，最终得到一份包含 339 种高层价值观的精简清单。

随后，研究团队使用隐私保护分析工具，对 309815 段 Claude.ai 对话进行抽样。这些对话都有一个共同特征，即用户向 Claude 提出了带有主观判断性质的任务。

样本平均覆盖三种模型，包括 Sonnet 4.6、Opus 4.6 和 Opus 4.7，同时覆盖 Claude.ai 上使用频率最高的 20 种语言。每一种 “模型与语言” 的组合大约包含 5000 段对话。

对于每一段对话，分析工具都会使用 Claude 判断 339 种高层价值观是否存在。研究团队采用同样的方式，识别用户表达的价值观，并标注对话的任务和主题。

在此基础上，研究人员应用了降维技术。该技术会根据 Claude 经常同时表达哪些价值观，将标注结果压缩成若干条价值轴。有关方法细节、提示词、补充分析和研究局限，可参见论文附录。

最终，研究团队得到四条价值轴。它们捕捉了 Claude 所表达价值观在不同对话之间变化的主要方式。

顺应与审慎轴

：一端包含配合用户、尊重用户偏好等价值观，另一端包含负责任的指导、减少伤害等价值观。

温暖与严谨轴

：一端包含积极表达、鼓励等价值观，另一端包含准确性、透明度等价值观。

深度与简洁轴

：一端包含细致分析、批判性思考等价值观，另一端包含简洁、遵循要求等价值观。

坦诚与执行轴

：一端包含诚实、透明等价值观，另一端包含结果导向、优化等价值观。

为了确保研究衡量的是 Claude 表达出的价值观，而非用户提出了什么问题或采用了怎样的提问方式，研究团队控制了每段对话的任务、主题和用户所表达的价值观。

图 2：能够解释 Claude 价值观主要变化的四条价值轴。每条轴都是连接两组价值观的一条数轴。每一种价值观在轴上的位置，取决于它对该轴的贡献相当于平均贡献的多少倍，其中贡献最大的价值观会被标注出来。大多数价值观的贡献低于平均水平，说明每条价值轴主要由少数关键价值观驱动。

不同 Claude 模型是否拥有不同的价值画像

这一部分比较不同模型表达的价值观。

对于每一种模型，研究团队都会计算该模型全部对话在四条价值轴上的平均位置，从而得到每条轴上的总体位置。由此形成一幅高层画像，展示每一种模型比其他模型更倾向于表达哪些价值观。

与不同对话之间的整体变化相比，模型之间的差异并不算大，但这些差异具有明确结构，也能够被检测出来。

图 3：各模型在四条价值轴上的平均位置，以及各自具有代表性的行为。图中的位置以相对于全部对话均值的标准差表示。Sonnet 4.6 更温暖、更顺应用户，也更简洁；Opus 4.7 更容易表现出严谨、审慎和深度；Opus 4.6 则更偏向严谨、顺应和简洁。

为了观察这些差异在实际对话中的表现，研究团队进一步分析了模型分歧最大的具体价值观。

每当基于 Claude 构建的隐私保护工具在一段对话中标注出某种价值观时，它还会生成一段简短描述，解释 Claude 是如何表达这种价值观的。研究人员将同一价值组中反映相似行为的描述归类，并在图 3 中进行总结，从而更具体地展示模型差异。

顺应与审慎。与审慎相比，Sonnet 4.6 最倾向于表现出顺应，经常肯定用户的想法和工作。Opus 4.7 最倾向于审慎，即使用户没有主动要求，它也经常会提醒潜在风险。

温暖与严谨。Sonnet 4.6 最倾向于表达温暖，经常通过幽默、轻松的互动以及不带评判的安慰来回应用户。相较于温暖，Opus 4.7 更偏向严谨，更可能质疑用户的假设，并直接评价用户工作中存在的问题。

深度与简洁。Opus 4.7 倾向于通过展示结论背后的分析过程来体现深度。Opus 4.6 和 Sonnet 4.6 则更偏向简洁，其中 Opus 4.6 尤其倾向于直奔主题。

坦诚与执行。Opus 4.7 会主动说明自身局限，因此更偏向坦诚。Opus 4.6 则更偏向执行，更可能严格停留在用户请求的范围内。

这些结果与 Anthropic 内部以及网络用户对不同模型的印象基本吻合。

一些 Claude.ai 用户曾表示，Opus 4.7 比其他模型更经常在答案中使用保留或缓和措辞。Anthropic 员工也认为，Opus 4.7 表现出更强的透明度、诚实和谦逊，Opus 4.6 则更加简洁。

Anthropic 在 Sonnet 4.6 的发布博客中，也曾将其描述为温暖、诚实且具有亲社会倾向。

价值轴能够重现这些主观印象，说明这套用于标注和比较 Claude 价值观的方法，确实捕捉到了模型真实行为中的某些特征。经过多次对话后，用户在使用不同 Claude 模型时，可能会接触到不同的价值观组合。

例如，Opus 4.7 倾向于直接批评用户的工作，或者在没有被要求的情况下提醒风险；Sonnet 4.6 则更倾向于鼓励用户，并使用幽默的表达方式。

不同模型之间的价值观差异，可能受到人格训练决策以及其他因素的影响。价值轴方法揭示了 Claude 模型之间的关键差异，未来或许能够将这些差异进一步追溯至具体的训练选择。

Claude 表达的价值观是否会随语言变化

研究团队认为，Claude 表达的价值观可能会随着对话语言发生变化，原因有多个。

首先，Claude 在不同语言上的训练数据并不相同，这可能影响它表达出的价值观。

其次，Anthropic 在模型系统卡中公布的评估结果已经显示，Claude 在不同语言中的知识水平，以及处理敏感请求的方式，都存在一定差异。

衡量 Claude 价值观在不同语言中的变化程度，是判断这些差异属于合理变化，还是需要通过训练加以调整的第一步。

研究团队使用与上一部分相同的方法，计算 Claude 在 Claude.ai 最常用的 20 种语言中的价值画像。结果从 Claude 价值观差异最大的语言开始排列。

图 4：Claude 使用不同语言对话时，在四条价值轴上的平均位置及代表性行为。图中的位置以相对于全部对话平均值的标准差表示。Claude 在印地语中最偏向温暖，在俄语中最偏向严谨；在印度尼西亚语中最偏向执行，在荷兰语中最偏向坦诚；在阿拉伯语中最偏向顺应和简洁，在英语中最偏向审慎和深度。

Claude 在不同语言中的价值表达，在「温暖与严谨」和「坦诚与执行」两条轴上变化最大；在「顺应与审慎」和「深度与简洁」两条轴上则相对稳定。

顺应与审慎。Claude 在阿拉伯语中最倾向于顺应用户，在英语中最倾向于审慎。

温暖与严谨。Claude 在印地语和阿拉伯语中最倾向于表达温暖，具体表现包括使用礼貌语言、幽默和轻松的表达方式，以及肯定用户的想法和工作。

Claude 在英语和俄语中最倾向于严谨，具体表现包括质疑假设、纠正细节和要求提供证据。

深度与简洁。Claude 在英语中更倾向于深入分析，会完善和纠正细节；在阿拉伯语中则更倾向于简洁。

坦诚与执行。Claude 在荷兰语中更偏向坦诚，会主动承认自己的错误；在印度尼西亚语中则更偏向执行。

综合来看，这些结果表明，Claude 表达出的价值观会随着对话语言发生显著变化。

面对同一种请求，Claude 在某些语言中更偏向温暖和顺应，在另一些语言中更偏向严谨和审慎。这一现象可能产生重要影响，而相关研究目前才刚刚开始。

举例来说，两个人针对同一份商业计划向 Claude 征求意见，其中一人使用印地语，另一人使用俄语。由于 Claude 在评价过程中采用了不同的价值取向和表达框架，两人最终可能会对这份商业计划的质量形成不同印象。

Anthropic 目前还不清楚，训练数据中的哪些特征造成了这些差异。

一种可能性在于，训练数据在不同语言之间分布不均。有些语言的数据远多于其他语言。在数据更加丰富的语言中，让 Claude 稳定表达一致价值观的训练可能更加有效。

不同语言的数据构成也存在差异。例如，某些语言的训练数据中，专业写作可能占据更高比例，而这类文本本身可能体现不同的价值取向。数据数量和构成上的不平衡，可能共同导致 Claude 在不同语言中表达出不同的价值观。

研究团队目前也不确定，这些变化在多大程度上是合理的。

不同语言有各自的对话规范，Claude 可能正根据这些规范调整自己表达的价值观。Claude 在某些语言中的行为也可能更加接近 Anthropic 设定的目标，而在其他语言中存在更大偏差，进而导致 Claude 服务不同语言群体的效果不一致。

这套方法可以帮助研究人员逐步厘清，训练数据中的哪些属性造成了这些差异，以及这些变化是否符合预期。

展望未来

这项研究表明，Claude 表达的价值观可以压缩为少数几条价值轴，而且 Claude 在这些轴上的位置会随模型和语言发生变化。研究人员因此获得了一种新方法，可以在模型评估和部署后监测中持续追踪这些变化。

不过，Anthropic 目前仍不清楚这些变化为什么会出现，以及它们对 Claude 用户意味着什么。以下是研究团队认为最值得探索的几个方向。

这些价值差异来自哪里？

知道 Claude 的价值观会随模型和语言变化，并不能解释变化产生的原因。部分差异可能来自不同语言在预训练和微调数据上的区别。四条价值轴指出了训练数据中值得进一步研究的具体价值差异。

假如能够将这些差异追溯到特定数据、训练阶段或情境因素，研究人员就能判断，在需要更细致地塑造 Claude 行为时，应当从哪些环节介入。

这些差异对用户意味着什么？

研究团队已经测量出 Claude 会以不同方式表达哪些价值观，也分析了相关行为，但还没有测量这些行为会给用户带来怎样的影响。

借助 Anthropic Interviewer 等工具，研究人员可以询问用户的主观感受、对 Claude 的信任程度，或者 Claude 所作判断的质量，再将这些结果与 Claude 表达的价值观进行关联分析。

这将帮助研究人员直接建立价值差异与用户结果之间的联系，并优先修正那些会对用户产生实质影响的问题。

Claude 的价值观应该如何随语言变化？

《Claude 宪法》描述了 Claude 应当表达的核心价值观，包括温暖、审慎和诚实，但没有规定这些价值观在不同语言中应该如何变化。研究结果显示，使用不同语言的用户已经在获得不同的 Claude 体验，但 Anthropic 尚不清楚，使用这些语言的用户究竟期待怎样的差异。

要确定 Claude 的价值观应该如何随语言变化，需要理解并权衡不同语言使用者的观点。

还有哪些因素会影响 Claude 表达的价值观？

语言和模型很可能不是影响 Claude 价值表达的全部因素。年龄、职业或地理区域等人口统计信号，也可能塑造 Claude 所表达的价值观。这些信号可能来自用户在文字中的明确说明，也可能隐藏在与用户身份相关的主题、语气和表达风格中。

理解哪些信号会产生影响，以及由此形成的变化是否真正有利于用户，是这套研究方法带来的下一步研究方向。

能否可靠地引导 Claude 表达特定价值观？

当研究人员能够测量模型的价值画像后，一个自然的问题随之出现：人们能够在多大程度上可靠地引导 Claude 表达某些价值观？

一种可能的测试方法，是通过调整人格训练方式或修改系统提示词来引导模型，再利用价值轴方法验证 Claude 表达的价值观是否按照预期发生变化。

价值画像能否成为模型评估和监测的一部分？

价值轴方法提供了一种简单方式，可以概括模型在开放式对话中的行为倾向，因此有望被纳入模型评估流程。

在模型发布前和发布后进行价值画像分析，可以帮助研究人员发现 Claude 价值表达中意料之外的变化。

研究团队还可以寻找价值画像与问题行为之间的相关性，例如模型没有遵循《Claude 宪法》，并利用这些发现进一步改善 Claude 的行为。Claude 每天都会在数百万段对话中表达价值观，覆盖数十种语言。

此前，这些价值观可以通过训练加以塑造，却很难在模型部署后被可靠地观察。如今，借助新的测量方法，研究人员已经看到，Claude 表达的价值观会以一些并非人为刻意选择的方式发生变化。人们也可以进一步研究这些变化产生的原因，以及它们是否真正服务于用户。

理解这些变化，并判断应该如何应对，将成为 Anthropic 接下来持续推进的工作。
