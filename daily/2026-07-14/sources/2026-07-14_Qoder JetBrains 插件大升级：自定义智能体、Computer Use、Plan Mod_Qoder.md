# Qoder JetBrains 插件大升级：自定义智能体、Computer Use、Plan Mode全来了

> 原文链接：https://mp.weixin.qq.com/s/HgcAktKLkf5kg6-Bf3DHTQ
> 公众号：Qoder

最近，Qoder JetBrains 插件迎来一波密集更新——自定义智能体、电脑控制、规模模式全部落地，对话体验也做了一轮优化。下面逐个拆解，看看这次到底升级了什么，以及怎么使用。

内置两大智能体：Computer Use 和 Plan Mode

除了自定义智能体，本次新版本还内置了两个实用智能体：

Plan Mode（规划模式）

——开启Plan Mode后， Qoder 会基于你的自然语言需求生成一份结构化的方案与规划。你可以先审阅和调整这份计划，再让智能体按步骤自动执行。

Plan Mode 有两种调用方式。一是自动调用——智能体模式会根据你的请求智能判断何时需要进行规划；二是显式调用——在对话框直接输入

/plan

加上你的任务描述，即可进入规划模式。

Computer Use（电脑操控）

——当任务涉及图形界面且无法通过命令行或 API 完成时，智能体可以直接操控桌面应用和浏览器来帮你完成工作 —— 而你可以继续在前台处理自己的事情。

Computer Use可以通过自然语言或者/computer-use命令唤起。

自定义智能体：你的工作流，你自己定义

除了内置的智能体外，现在你可以直接在Qoder JetBrains 插件里创建、配置和调用自己的智能体。

比如你团队有一套特定的代码审查流程，或者你习惯用某种固定 prompt 处理某类任务，你可以直接在对话框输入

/create-subagent

加上你的需求描述来创建智能体。

Qoder 内置了 create-subagent 技能，会通过交互式引导帮你快速创建一个符合规范的自定义智能体，不用手动去配置页面填参数。

对话体验优化：支持插话（Steer）

以前 Qoder 在执行任务的时候，如果你想补充方向或修正要求，只能等它做完才行。现在Qoder JetBrains插件支持插话（Steer）——任务执行过程中可以随时补充方向或修正要求，不用中断重来。

同时消息队列的展示也做了优化，多条消息的排列和加载逻辑更清晰，不再出现消息乱序或堆积的情况。

Qoder Cleanup：一键清理旧文件

点击菜单栏的

Tools > Qoder > Qoder Cleanup

可以一键清理过时的 Qoder 相关文件。插件迭代过程中可能残留旧版本配置或缓存，这个功能帮你一次搞定，不用自己去翻目录手动删。

打开JetBrains IDE，升级到Qoder JetBrains插件2026.709.1即可体验。
