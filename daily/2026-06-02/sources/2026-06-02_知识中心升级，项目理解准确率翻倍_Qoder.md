# 知识中心升级，项目理解准确率翻倍

> 原文链接：https://mp.weixin.qq.com/s/OD9LUdtZO6bQdvza1MnhwQ
> 公众号：Qoder

1 images

Qoder Desktop v1.5.2 发布，知识中心全面升级，Agent 对项目的理解更准、更深。

知识中心包含三类知识来源：

1、Repo Wiki（为你和团队准备）: 基于代码库自动生成项目文档，覆盖架构、模块关系与实现细节，代码变更后自动同步，通过 Git 共享给团队。

2、知识卡片（为 Agent 准备）: 与 Repo Wiki 一起生成，包含架构文档、代码规约和技术栈信息。每次 commit 自动追踪变更，Agent 的知识始终与代码同步。

3、记忆（Memory）: 对话中的踩坑记录、决策依据和项目经验自动提炼并长期保存，支持主动记忆。用得越久，Agent 对项目理解越深。

三类知识汇入同一个引擎，每次 commit 自动更新，每次对话自动学习沉淀。新增 /knowledge 内置智能体，在对话框中即可读取或编辑 Wiki 文档和知识卡片。

92 个真实场景实测，有了全新整合后的知识引擎，Agent 找到正确项目信息的准确率从 40% 提升到 77%。

Qwen3.7-Max 每天免费 200 次模型调用仍在继续，搭配使用表现更好：知识中心提供正确的项目上下文，Qwen3.7-Max 提供强推理和代码生成能力，Agent 更懂项目，交付质量更高，表现完全不输全球顶尖模型。

v1.5.2 还有以下更新:

- 语音输入支持智能润色，语音转文字后自动优化表达

- 大型仓库下 Repo Wiki 与知识卡片的生成与更新性能优化

- 代码变更场景下的知识命中效率提升

- Quest 视窗下的智能体模式效果与体验优化

- 知识使用流程优化，降低补充项目背景信息的操作成本
