# Agentic 时代的技术地图：AICon 2026 上海站全日程正式发布

> 原文链接：https://mp.weixin.qq.com/s/DvYuVKYDOW53AEcsD_O9rQ
> 公众号：InfoQ

当 AI 不再只是工具，组织本身开始被重塑。

过去一年，"Agent"这个词从实验室走进了生产环境。工程师们开始真正面对一个新的问题：不是"AI 能不能做到"，而是"我们能不能把它跑稳、跑对、跑出规模"。架构怎么设计？记忆怎么管理？多智能体之间如何协调？研发团队的工作方式又该如何重构？

这些，正是 AICon 2026 上海站试图回答的问题。

AICon 全球人工智能开发与应用大会面向资深工程师、技术管理者与企业决策者，是国内最具技术深度的 AI 工程实践盛会。今年，大会以"构建可信赖、可规模化、可商业化的 Agentic 操作系统"为核心命题，横跨 6 月 26 日（周五）至 27 日（周六）两天，集结来自阿里、腾讯、蚂蚁、字节、快手、小红书、华为、Google Cloud 等数十家头部公司的技术专家登台分享。

1

四场主题演讲，四个产业级判断

今年大会邀请到四位来自学界与产业界的重量级嘉宾发表主题演讲，覆盖多模态、云原生基础设施、大模型生产力落地与 Token 服务体系四个关键维度。

邱锡鹏｜复旦大学教授 & 模思智能创始人

MOSS 系列开源大模型主持研发者，论文引用超 4 万次，2024 年 CCF-ACM 青年科技奖得主（全年仅 1 人）。此次将带来《迈向交互新范式——MOSS 多模态模型的创新与实践》，系统介绍 MOSS-VL、MOSS-Audio、MOSS-TTS 三个方向的最新进展，以及长视频、实时语音等场景下的推理效率优化路径。

张献涛（旭卿）｜阿里云智能集团副总裁

神龙计算架构主导发明者，国家科技进步二等奖获得者，Xen/KVM 等国际顶级开源项目早期核心贡献者。此次演讲《From Cloud Native to Agent Native，当云的服务对象从人变成 Agent》将直面一个产业级的关键判断：阿里云服务数千家企业客户的实践中，超过 70% 的 Agent 工程挑战不在模型层，而在执行层。Serverless 十年的无状态假设，正在被 Agent 的有状态本质所挑战。

周俊｜蚂蚁集团副总裁

国家级创新领军人才，H-index 52，顶会论文百余篇，阿里和蚂蚁 AI 创始成员之一。此次将发布百灵 v2.6 系列模型并分享系统性技术探索，重点讲清楚大模型如何从"会回答"走向"能执行"——从 Agent 能力、推理效率到 Async RL 训练范式，一套完整的工作流智能构建路径。

翟季冬｜清华大学教授 & 高性能计算所所长

国家杰青获得者，指导清华学生超算团队十五次获得世界冠军，清程极智首席科学家。此次演讲《迈向 Token 服务新范式——从评测到调度的全链路优化供给

》将从评测体系、智能调度到推理引擎协同优化，拆解 Token 经济爆发背景下，规模化服务的全链路工程实践。

2

两大命题，构成大会内容主体

今年大会超过一半的专题演讲，集中在两个核心命题上。

命题一：Agent 怎么真正用起来

如果说 2024 年大家还在讨论 Agent 的可能性，现在这个问题已经不重要了——真正的战场，是生产环境里那些没有教科书的工程难题。

围绕这个命题，大会设置了工程化、记忆与运行时基础设施、安全与可信治理三个专题，合计十余场演讲，覆盖 Agent 落地的三个核心维度。

工程化方向，火山引擎裴斐的《Agent Mesh：企业多智能体系统治理实践》直面多 Agent 协作最棘手的治理问题；阿里云王炳燊从沙箱网络安全切入，讲清楚 Agent 运行时的流量隔离与智能治理；科大讯飞王搂则带来超大规模教育场景下的 Agent 系统架构演进完整案例。

记忆方向，盛大集团邓亚峰的《基于长期记忆系统 EverOS 的自我演进》直指 Agent 能否真正"记住"并持续进化；MemVerge 范承工带来面向智能体与人类用户的 AI 记忆系统架构设计与核心场景实践；火山引擎马进则分享从多模态数据湖到 Agent 记忆湖的演进路径。

安全方向，上海人工智能实验室、阿里 AAIG 红队、蚂蚁安全团队分别从智能体操作系统安全、自动化红队实践、Agent 安全漏洞检测三个视角，回答同一个问题：Agent 规模化落地，安全底线怎么守？

命题二：研发组织怎么被 AI 重塑

AI Coding 工具普及之后，真正的挑战才开始。这届大会围绕研发重构设置了两个专题，从不同角度切入同一个问题。

企业级研发体系的重构专题，关注的是组织层面的系统性变革。蚂蚁数科的演讲从 SDD 规范驱动、Harness 工程实践到 AGI 研发体系重塑，提供了一套企业级 AI 原生研发转型的完整样本；菜鸟郭凤钊的《从 Vibe Coding 到需求托管交付 Agent》，则把视角从个人工具延伸到了交付闭环；小红书郑鑫祺将带来 Vibe Coding 平台从 0 到生产级别的高可用架构实践。

AI 开发生产力重构专题，则更聚焦工程师个体的工作方式变化。快手李京讲的是从个人提效到组织生产力提升的演进路径；阿里徐亮亮的《Qoder CLI：终端里的全栈工程师》，探讨 Coding Agent 如何成为企业级 AI 应用基础设施。

这两个专题合在一起想回答的，是研发组织在 AI 时代该如何重新运作——不只是"用了 AI"，而是从工具到范式的真正转变。

其余专题：技术广度的另一面

AI 原生数据工程

：数据架构如何跟上 Agent 的节奏，从算力调度到存算协同，回答 AI 时代数据基础设施的新挑战。

世界模型与多模态智能 & 大模型推理优化

：腾讯混元世界模型、阶跃星辰多模态生成与理解架构演进，代表大模型能力的前沿突破方向；京东 xLLM 投机推理、华为 Omni Cache KV 管理优化、阿里云记忆感知 KVCache 优化，则聚焦推理效率，回答"模型跑得更快更省"这个生产环境里的真实诉求。

金融领域大模型落地实践

：金融反欺诈、银行信贷全链路变革、保险智能风控、内容安全治理——金融是大模型落地最严苛的场景之一，这个专题集中呈现头部机构的真实工程经验。

企业智算架构弹性适配

：蚂蚁集团训推一体潮汐弹性调度、快手推荐系统池化探索、阿里云超大规模集群调度与容错实践——算力不只是买多少 GPU 的问题，怎么用好、用稳才是硬功夫。

AI 时代的移动端开发范式 & 端侧智能

：华为鸿蒙跨平台框架演进、飞猪跨端技术实践、面壁智能端侧大模型技术趋势、OPPO 多模态 Agent 手机交互重构、涂鸦智能本地执行与全场景硬件接入——从移动端开发框架到端侧 AI 落地，回答"AI 的最后一公里在端侧怎么走"。
