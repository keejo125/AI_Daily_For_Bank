# GPT-6要来了！OpenAI彻底抛弃4T旧底座

> 原文链接：https://mp.weixin.qq.com/s/aflEagsmfrs_2FPQATpaig
> 公众号：新智元

新智元报道

【新智元导读】

OpenAI紧急提档GPT-6，彻底抛弃4T旧底座？而且，GPT-5.6正式解禁，周四就要上线！Sol、Terra、Luna三款模型全面开放，编码、科研、日常任务全覆盖。已经有早期测试者狂赞Sol模型「永不放弃」，现在，全体开发者已经在摩拳擦掌了。

就在刚刚，一条消息瞬间传遍圈内。

GPT-6的发布时间大幅提前，最快将于本月内正式空降！

如果爆料属实，就意味着GPT-5.x系列将以GPT-5.6收官，成为绝唱。

据悉，这次OpenAI做出了一个极具魄力的决定——彻底抛弃之前约4T参数量的「Spud」旧底座，转而押注规模更庞大的全新预训练底座。

显然，OpenAI的意图非常直接：用这个全新底座，正面硬刚Anthropic即将在几周内发布的Fable 5.1。

而且就在昨天下午，好消息传来：GPT-5.6全面解禁！周四见！

昨天中午，Axios独家爆出猛料：官方已正式解除对 OpenAI 的 GPT-5.6 访问限制。

紧接着，OpenAI在X上官宣：旗舰模型 GPT-5.6 Sol 连同 Terra 和 Luna，将于本周四（7月9日）面向全球用户正式公开发布！

被寄予厚望的「王炸」模型，艰难度过一个月的「政企特供有限预览」后，终于被解禁了。

奥特曼对开发者们喊话：Happy Building！

逼宫、谈判与深夜解禁

6月26日，OpenAI低调开启过GPT-5.6的预览，仅限少数人。

因为美国商务部下属的「AI 标准与创新中心」直接入驻了 OpenAI，甚至把 OpenAI 的技术专家强行留下，进行「安全审查」。

根据此前的最新 AI 行政令，前沿模型的发布必须经过实时的case-by-case 谈判。

现在，为什么官方突然松口了？

业内人士透露，OpenAI 的专家在华盛顿经历了无数次真实攻击压力测试与漏洞修复，最终成功证明：GPT-5.6是OpenAI 迄今为止最强健的安全系统。

OpenAI 在今天的公告中，特别话里有话地强调：「我们与官方合作，但短期限制是为了更广泛的可用性铺路，我们绝不希望官方提前访问成为常态，技术不能被垄断。」

现在，靴子落地，官方亮起绿灯。等待已久的全球开发者社区，期待值瞬间拉满。

「三箭齐发」：Sol、Terra、Luna 到底有多强？

这次的三款全新模型，分工明确。

GPT-5.6 Sol：最强旗舰

Sol主打前沿 frontier 能力，专攻复杂编码、前沿科学和网络安全任务。

它最大的特点，就是极限推理。

有人说，GPT-5.6的能力提升，堪比GPT-4/o1级别的飞跃。

早期测试者 Max Weinbach 激动地评价：「这款模型最恐怖的地方在于，它永不放弃！如果你把它丢进极限推理模式，它会疯狂死磕，不把问题彻底解决绝不停手。」

还有人说，GPT-5.6极其执着，即使不用/goal命令，也能运行一整天。

GPT-5.6 Terra：性价比之王，5.5 终结者

性能无限接近此前的超级旗舰 GPT-5.5，但成本直接砍掉一半！

这是专门为了日常高频工作量身定制的打工神器。它的出现，意味着GPT-5.5已经out了。

GPT-5.6 Luna：快如闪电，便宜到发指

Luna专注于极致的速度和极低的成本，是高频调用Agent的首选。

更夸张的是硬件生态的无缝大咬合。据悉，GPT-5.6 Sol将直接登陆Cerebras推理平台，其推理速度达到了令人窒息的每秒750 tokens！

在写代码、生物医药研发、网络攻防等高强度场景下，这体验简直秒杀。

难怪有提前测试的开发者感慨：「当我无法使用5.6版本时，我感觉自己快要疯了。」

就在今天，Codex CLI 0.143.0 版本的更新日志也意外泄露：Amazon Bedrock 平台已经悄悄添加了 GPT-5.6 三剑客，并且对 Max 推理强度提供了支持。

从配置来看，这批模型极有可能是专为Max档准备的。

第一批实测的大佬，发话了

现在，X上提前拿到实测资格的网友们已经开始点评了。

知名开发者 Theo 连发数条X，直接吹爆。

GPT-5.6 它解决了我在使用 5.5 时遇到的所有痛点。它极其执着，即使你不给它下达 /goal指令，它也能自己运转一整天！它对子代理的理解和编排能力出神入化。 它让我使用电脑的频率提高了百倍，当内测结束、我无法再使用 5.6 的时候，我感觉自己快要疯了！

Magicpathai CEO Pietro Schirano也给出好评。

毫不夸张地说，这是我用过的最好的模型。速度快、极具创意，而且他们终于把前端设计问题给解决了！我已经整整两个月没有自己检查过写的代码了！

沃顿商学院教授Ethan Mollick，则把 Sol 拿来和Fable对比。

Sol 的能力和 Fable 类似，但体感完全不同。Fable 喜欢按照自己的节奏跑去独立完成任务；而 Sol 速度极快，更喜欢一步一步陪着你高频互动。 当我还没想清楚自己要什么的时候，我会用 Sol 来回碰撞；当我能明确定义任务时，我会用 Fable 跑长线。如果遇到地狱级难题，我会上 Sol Pro。

而 AI 创业者 Matt Shumer 却唱了点反调，他直言：

Sol 是个很棒的模型，但在我测试的绝大多数任务里，Fable 依然更好，而且更具 Agent 属性。

LMArena 掌门人终极定调：睿智猫头鹰和夺命罗威纳

作为重量级玩家，LMArena 负责人 Peter Gostev 也放出了目前全网最硬核的深度对比。

他给出了一个极具视觉冲击力的比喻：Fable 5 像一只「睿智的猫头鹰」，而 GPT-5.6-Sol 就是一只「罗威纳犬」——它会死死咬住问题的咽喉，直到彻底解决绝不松口！

在 Gostev 看来，两者的底层逻辑和体感有着强烈的冲突。

首先，Fable 5显得更傲慢。

Fable 本质上更聪明、洞察力更深。但在构建新基准测试时，Fable 尽显「傲慢」——40分钟跑完，虽然听起来很智能，但只是凭感觉草草了事，还自导自演给自己打100分。

而 GPT-5.6-Sol 极其勤奋，花了整整两天才死磕出一个经过全面测试、运行良好的完美基准。

相比之下，GPT-5.6-Sol是个干活机器，安全感拉满。

虽然 Fable 在从零手搓 UI 和文本写作上依然碾压，但它总会遗漏关键功能。

GPT-5.6-Sol 则带来了无与伦比的安全感——丢给它 8 个待办事项，它能不折不扣地全干完，绝不漏项。配合强大的

/goal

指令，Sol 可以连续高频运行几天几夜。

此外，Sol 在遵循现有代码模式、视频剪辑（可无缝处理1小时素材）、管理子代理通讯、深度研究以及 Token 效率上，均实现了质的飞跃。

Gostev 甚至分享了一个极其真实的「脏话计数器」：在 GPT-5.5 时代，他写代码时对 AI 爆粗口的频率是 4-5%；用了 5.6-Sol 后骤降到 1-2%；而一旦切回 5.5，暴躁指数瞬间飙升到 7%！

总之，单论智商，Sol 确实不如 Fable。但它是一个不知疲倦的超级干活机器，不会对你说教。

由此，Gostev 给出了当前大模型的最强工作流解法：

用 Fable 进行架构讨论和头脑风暴 → 丢给 GPT-5.6-Sol 去疯狂手搓代码、执行脏活累活 → 最后再用 Fable 编写文档和对外沟通。

就在明天，GPT-5.6三款模型会正式营业。

这一次，轮到全球开发者亲自上手打分了。
