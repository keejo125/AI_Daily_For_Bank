# Codex新升级：彻底打通Windows生态！

> 原文链接：https://mp.weixin.qq.com/s/Yayf2DXWrvw9zFxoU61Cpw
> 公众号：InfoQ

2 images
昨日，每周四的Codex活动破例改期至周五举行。不少人猜测，新模型GPT-5.6将会在今天发布。原因之一是，在Codex 后端日志中已出现 GPT-5.6。
然而，几个小时后，OpenAI仅发布了Codex 的又一次更新。有知情人士在X上表示，GPT-5.6 的开发工作仍在继续，今天早些时候，内部刚发布了一个性能显著提升的新检查点。
此次Codex的新功能专为Windows 用户们而来。现在，Codex 已支持 Windows 电脑操作，用户可直接在 Windows 电脑上执行各类操作。同时，ChatGPT 移动端应用中的 Codex 现已适配 Windows，用户可以随时随地启动、查看并操控任务，而相关工作会持续在 Windows 电脑上运行。
同时，Codex 已进一步适配 Windows 开发者全流程。依托 Windows 端的计算机操控能力，Codex 可直接在项目环境中完成应用测试、流程调试与代码审查。
目前，这项升级尚处于早期体验阶段，OpenAI正持续开发更多功能。
