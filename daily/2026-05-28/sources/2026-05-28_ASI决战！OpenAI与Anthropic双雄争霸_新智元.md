# ASI决战！OpenAI与Anthropic双雄争霸

算力帝国 vs 企业刺客，OpenAI与Anthropic的ASI终极对决已经开打。

几周前，AI江湖出现了一个标志性事件。

那个

曾经

天天挥舞着Grok、扬言要挑战OpenAI的马斯克，突然把自己手里最醒目的算力王牌——Colossus 1，租给了Anthropic。

这是一个装着超过22万张NVIDIA GPU的怪兽级集群，是马斯克此前用来撑起xAI野心的钢铁心脏。

现在，它开始为Claude跳动。

就在今年2月马斯克还在X上公开骂Anthropic是「Misanthropic」（反人类）、「evil」（邪恶）。

戏剧性的是，三个月后他的态度来了个180度大反转。

他花了一个星期和Anthropic核心团队深聊，然后告诉世界：「没人触发我的『邪恶探测器』。Claude大概率是好的。」

骂归骂，生意归生意，更要紧的是——敌人的敌人就是朋友。

马斯克在法庭上向OpenAI索赔1500亿美元、要求罢免Sam Altman。他此刻把超算租给的，恰恰是Altman最大的对手。

这操作真是一箭三雕：赚钱、盘活闲置资产、强化OpenAI的对手。

但更加重要的是，这是一个更加宏大事件的标志性时刻。

从这一刻起，ASI世界的终极对决，正式收敛成双雄争霸之势——一个手握ChatGPT、Codex、GPT-5.5和Stargate帝国的OpenAI；和一个手握Claude、Claude Code、Mythos和AWS/Google/SpaceX多云算力的Anthropic。

ASI前夜，两家公司的决战已然开启。

本篇为新智元ASI产业图谱4月号文章，本文我们再次聚焦ASI的最新进展，一起探寻ASI的最深入洞察。

两大超级生态的力量版图

是OpenAI最老牌的盟友，累计投入超130亿美元，把GPT深度嵌入Azure、Office、Copilot全产品线。

8520亿美元

5000亿美元

10GW

此外，OpenAI还组建了一支19家咨询机构组成的「部署军团」，专门帮企业落地Codex编码工具，直接贴身肉搏Anthropic的企业地盘。

如果说OpenAI走的是「帝国模式」——一个主权，一支军队，一面旗帜，那么Anthropic走的就是「瑞士银行模式」：

不跟任何人独家绑定，让所有人都觉得投它是最安全的对冲。

亚马逊

谷歌

OpenAI的核心盟友

Claude成了全行业唯一同时跑在AWS Bedrock、Google Vertex AI和Azure Foundry三大云平台上的前沿模型。

再加上刚到手的马斯克Colossus 1——

5月19日，又一颗重磅炸弹落地：

OpenAI联合创始人Andrej Karpathy官宣加入Anthropic。

这位「Vibe Coding」的发明者、250万粉丝的AI教育家、前特斯拉自动驾驶负责人，把自己创立的教育公司按下暂停键，直接进了Anthropic的预训练团队，做一件非常具体的事：

用Claude加速训练下一代Claude。

从2024年Jan Leike、John Schulman，到2026年Andrej Karpathy，三位OpenAI前核心人物先后转向Anthropic一——顶级AI人才正在用脚投票。

实力硬碰硬：谁在赢？

营收：Anthropic上演人类商业史最快增长

Anthropic的年化营收已经超过300亿美元，Q2单季预计达到109亿美元，甚至可能首次实现季度运营盈利。四个月前，这个数字还是90亿。

从90亿到300亿，四个月，这是人类商业史上最快的营收增速，没有之一。

反观OpenAI，年化营收大约在240-250亿美元区间——仍然强劲，但增速已经被Anthropic甩开。

ChatGPT坐拥近9亿周活用户、5000万付费用户，消费端的统治力毋庸置疑。但消费端的问题是：用户多，客单价低。

Anthropic没有那么多用户，但它吃的是企业预算——最贵的那一块。

更耐人寻味的是效率差距。2025年OpenAI的训练成本约200亿美元，Anthropic只花了50亿——四分之一。

Anthropic用OpenAI四分之一的训练成本，跑出了比它更高的营收。OpenAI预计2030年才能盈亏平衡，Anthropic的目标是2028年——快了整整两年。

34.4%

32.3%

。一年前这个数字是8% vs 32%，差距看起来不可逾越。

更致命的是：当企业第一次选择AI供应商时，

70%选了Anthropic

。

42-54%之间，OpenAI只有21%

25亿美元

但这恰恰证明了一件事：

程序员已经离不开它了。

模型与产品：全栈巨兽 vs 企业刺客

模型层面，两家已经打出了完全不同的牌面。

OpenAI的王牌是GPT-5.5——定位为面向软件工程、科学研究和复杂专业工作的全栈模型。

配合Codex（每周超400万用户）和与Dell合作推出的混合云方案，OpenAI想做的是「什么都能干」的AI全家桶。

Anthropic走的是完全不同的路线。Claude Opus 4.7主打自主、长时程Agent工作和深度知识处理；Claude Code在SWE-bench上拿下80.8%的全行业第一；Mythos则直接切入网络安全——在「Project Glasswing」计划中发现了超万项高危漏洞，欧洲央行罕见召集银行加速修补，金融稳定委员会要求获取简报。

而刚曝光的Claude Memory Files——一种类似个人Wiki的文件化记忆系统——则是在为下一代长期运行的AI Agent铺地基。

一句话：

OpenAI在做超级应用，Anthropic在做企业操作系统。

一个想让全人类每天打开ChatGPT，一个想让每家公司每天离不开Claude。

ASI何时到来？

一切竞争的终极悬念是：

超级智能（ASI）什么时候来？

AI安全评估机构METR的数据提供了最直观的参照系：AI自主完成编程任务的「时间视界」（即AI可以独立完成多长时间的人类专家任务）正在指数级增长——

2019年（GPT-2时代）：3秒

2025年底（Claude Opus 4.5）：近

5小时

倍增速度：从每7个月翻一倍，加速到每

4个月

翻一倍

Claude Mythos Preview：已经接近16小时

换句话说，AI 的「可独立工作时长」不是线性变长，而是在按指数级撕开人类任务的时间边界。今天是5小时，明天就是一天、一周、一个月。

另一边，AI Futures Model的最新Q1 2026更新，将「自动编码者」（Automated Coder，即AI完全替代人类程序员的节点）的中位预测从2029年底前移到了

2028年中

。

而从自动编码者到ASI的「起飞期」，模型给出了30%概率在1年内完成的估计。

换句话说：

按当前趋势，2028-2030年间出现ASI并非天方夜谭。

这正是Karpathy放弃自己的公司也要跳进Anthropic预训练团队的原因——

用

AI

训练下一代AI

。

也是Anthropic年薪40万美元疯抢文案、OpenAI出价44.5万美元专招「AI递归自我改进安全研究员」的原因——两家都知道，他们正在建造的东西，

可能是人类造出的最后一项技术。

METR在2026年2-3月的前沿风险报告中已经发出警告：在这两个月里，AI Agent在AI公司内部已经以与人类员工相当的权限和监督水平，自主运行在真实的研究与工程项目上。部分Agent的活动甚至没有任何人类审查。

棋局已经进入中盘。

OpenAI用5000亿美元的Stargate押注「算力即一切」；Anthropic用四分之一的成本、三朵云的灵活度、70%的企业首选率和一连串从OpenAI来的核心人才押注「效率×安全×信任」。

Colossus 1的机房灯通宵不灭。只不过现在，那22万张GPU日夜燃烧的，不再是马斯克的野心，而是人类通往ASI的最后一段路。
