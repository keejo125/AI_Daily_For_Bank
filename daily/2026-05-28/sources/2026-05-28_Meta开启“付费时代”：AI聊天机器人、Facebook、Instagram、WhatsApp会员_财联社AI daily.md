# Meta开启“付费时代”：AI聊天机器人、Facebook、Instagram、WhatsApp会员都来了

> 原文链接：https://mp.weixin.qq.com/s/YQLs0JmFT-NeyZNFG8wnvA
> 公众号：财联社AI daily

文|

史正丞

狂砸上千亿美元投资AI后，全球最大社交媒体云供应商Meta终于向股东们展示了赚钱的欲望。

公司周三宣布，

将从下个月开始测试Meta AI付费订阅服务

，同时还将

推出面向企业和网红的订阅套餐

。对于普通消费者，Meta旗下三大平台也将推出

每月付费3或4美元的订阅服务，提供一些个性化功能

。

受此影响，Meta股价盘中拉升，最终收涨近4%。

AI订阅服务亮相

虽然Meta目前并不以拥有强大AI模型著称，但新的AI订阅服务已经摆上台面。

公司披露，新的AI订阅套餐分为两档。对于频繁使用Meta AI生成图像和视频，或者进行深度推理的用户，将推出一档

月费7.99美元的Meta One Plus订阅服务

；更高级别的套餐名为Meta One Premium，

每月费用为19.99美元

，包含与Meta One Plus相同的功能，但使用额度更多。

与主流竞品类似，用户仍可免费使用Meta AI聊天机器人，但使用量会受到限制。公司拒绝透露各等级的具体使用上限，仅表示Premium套餐的额度会显著高于Plus套餐。

对于一家本财年将投入至多1450亿美元，未来数年将在AI基建上至少砸下6000亿美元的公司而言，开始直接用AI服务赚钱将是重要的业务进展。

目前，Meta超过98%的营收都来自于出售社交媒体流量广告。

公司表示，未来几周内这些AI计划还将扩展，为使用AI眼镜的用户提供更多福利。该计划将于下月开始测试，首先在新加坡、危地马拉和玻利维亚进行。

除了AI外，Meta还将于本周晚些时候开始测试面向创作者和企业的两项订阅计划。其中，每月14.99美元的Meta One Essential 计划将提供认证徽章、防冒充保护等功能，与多数社交媒体平台的认证服务类似。

更昂贵的Meta One Advanced计划（每月49.99美元）将包含Essential计划的所有内容，此外还能

在Facebook信息流中获得推荐位、在搜索结果中排名更靠前、并自动向与该用户内容互动的人发送“关注”邀请。

个人订阅服务同步上线

对于每月使用Meta旗下各大社交媒体的近40亿地球人，Meta也端出了赚钱计划。

公司周三确认，

对Facebook、Instagram和WhatsApp均推出Plus订阅服务，前两个平台每月3.99美元，WhatsApp Plus定价2.99美元/月。

（Meta产品主管Naomi Gleit宣布推出订阅服务）

3个订阅计划分别针对各应用量身定制：Facebook Plus和Instagram Plus更侧重社交表达功能，而WhatsApp Plus则聚焦个性化设置与消息服务。简单来说，就是花钱让用户账户看起来与众不同。

Meta订阅业务负责人Helen Ma表示，用户可以单独购买每个APP的订阅服务，也能打包一起订阅。
