# Qoder Mobile 上线：开启新的工作方式

> 原文链接：https://mp.weixin.qq.com/s/xGHyuIWHOQtizYON_WO-yw
> 公众号：Qoder

Qoder 是全平台覆盖的 AI 编程助手。手机不只是辅助——它具备桌面端的大部分能力，还能远程向电脑发起任务。鸿蒙版体验与 iOS、Android 完全一致，从桌面到命令行到手机，同一个任务、同一份上下文，无缝衔接。

随时随地发起，跨端同步

Qoder App 让你的手机不只是辅助工具，手机上具备桌面端的大部分能力，随时随地打开 App 就能开始工作。任务在桌面端同步运行，你在手机上发起的工作，回到电脑前直接看结果，同一个任务、同一份上下文，跨端无缝衔接。

用手机开启新的工作方式

我们开始使用移动端处理各种事务：查看运行状态、审批 Agent 提交的方案，并在方向不符合预期时及时停止或补充指令。

你还可以在手机上直接向桌面端或命令行远程发起任务——代码重构、测试跑批、问题排查、环境部署，一条指令就能启动。

这也让一些过去必须回到电脑前才能处理的事情，有了新的工作方式：

及时处理故障

如果你在午餐或通勤途中收到告警，可以直接用手机启动 Agent，让它先检查日志、排查问题，并给出可能的修复方案。

等你回到电脑前，初步定位和修复工作可能已经完成，可以直接进入结果审查，缩短从收到告警到开始处理之间的等待时间。

反馈直接变成任务

当你群聊中看到用户反馈时，可以直接截图并添加标注，再把图片作为上下文交给 Agent。

发现问题的当下，就可以让 Agent 开始处理，不必等回到电脑前再重新整理和描述。

随时了解进展

任务运行过程中的关键节点，Qoder 会使用 iOS 灵动岛和实时活动（Live Activity）推送告诉您——我们完成了一个阶段、遇到需要你确认的决策点、或者任务跑完出了结果，你都会第一时间知道。通知内容清晰可读，不用点进去也能快速判断下一步。

接下来

我们会持续迭代 Qoder Mobile，让云端运行 Agent 的体验逐步接近在本地电脑上运行。

任务将支持连接 GitHub 或你的 Workspace，并在云端持续执行。即使关闭电脑，任务也不会中断，运行更加省心。

在此之前，我们会继续优化远程控制能力，以及本地任务与云端任务之间的衔接，让你可以根据不同场景灵活选择任务的运行环境，同时保持任务进度与上下文的连续。

我们也会进一步完善手机端的成果审查体验。任务产物和代码 Diff 都可以直接在手机上查看，不必回到电脑前，也能快速判断 Agent 的产出是否符合预期。
