# 你的 Agent，终于不用手把手教了

> 原文链接：https://mp.weixin.qq.com/s/l5fuFHZpotY9JGplc_tCfQ
> 公众号：Qoder

1 images

如果 Agent 只能处理文本、只能被动等指令，那它再聪明也只是个聊天框。你要的是能干活的搭档，不是要全程教的工具。

我们接到很多用户有明确的需求，所以这次我们也做了相应的更新和优化。

一、能看图了，也学会主动干活了

对话里直接生图、以图搜图——做方案要配图、拿竞品截图找参考，不用再跳出去开别的工具。

更关键的是事件驱动。任务跑完、跑出错，Agent 自己知道，自动把结果推进群里、把异常写进日志。不用你盯着，也不用轮询。从「你问它才动」变成「事情一发生它就动」。

二、能一口气吞下一堆信息了

上下文窗口从 200K 到 1M 可选，按场景适配。几万行代码 review、一整季会议纪要汇总，一次性丢进去，不用再分批喂。

文件上传也扩到了图片、压缩包、音视频。竞品录屏丢给它提取交互流程，开完会把录音传上去直接出纪要，省掉手动转文字这一步。

三、搭 Agent会更快了

技能中心支持多选和批量导入，搭一个多能力 Agent 从逐个配变成几轮点击的事。Session 里 Skill 和 MCP 标签能点开看详情，MCP 服务器还新增了「交互策略」选项。

交互也打磨了：对话框换成 ChatGPT 风格的发送/停止切换，长指令想中途打断不用满屏找停止按钮；Session 详情页新增 Agent、Environment、Vault 分页，关联资源一目了然。

其他还做了一些体验的优化——

1. Session 输入框改为 ChatGPT 风格的发送/停止切换按钮，操作更直觉。

2. Session 中的 Skill / MCP 标签支持点击查看详情。

3. MCP 服务器配置新增「交互策略」选项。

4. 技能中心支持多选 + 一键批量导入，快速装配 Agent 能力。

5. Session 详情页新增 Agent / Environment / Vault 分页，关联资源一目了然。

这几件事放一起，就一个目的：让 Cloud Agent 从被动应答的聊天框，变成能感知事件、能处理多模态信息、能主动行动的协作伙伴。不是你手把手教它，是它自己知道该干嘛。
