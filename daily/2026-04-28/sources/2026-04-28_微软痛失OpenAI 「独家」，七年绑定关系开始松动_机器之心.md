# 微软痛失OpenAI 「独家」，七年绑定关系开始松动

> 原文链接：https://mp.weixin.qq.com/s/siSPjiPQknxdcnaqRCdT5g
> 公众号：机器之心

机器之心编辑部

一夜之间，微软对 OpenAI 技术的独家地位被打破。

就在昨晚，双方达成全新的合作协议：

依然保留微软作为 OpenAI 的主要云合作伙伴，微软对 OpenAI 模型和产品的知识产权许可将变为非独占，有效期维持到 2032 年不变

。

在去年 10 月，OpenAI 才与微软签署了一份

协议

：

微软支持 OpenAI 董事会推进成立公开利益公司（即 OpenAI Group）和再资本化。在再资本化之后，微软将持有 OpenAI 大约 27% 的股份，总估值高达约 1350 亿美元。

受此消息影响，微软股价周一开盘下跌约 1%，而在盘前交易中一度跌幅约 4%。

Sam Altman 发推称，「我们已经更新了与微软的合作关系。微软仍然是我们的主要云合作伙伴，但现在我们可以在所有云平台上提供自己的产品和服务。同时，我们会继续向微软提供模型和产品，合作期限到 2032 年；并在 2030 年之前维持收入分成安排。」

这一调整也为 OpenAI 将模型直接部署到 Amazon Web Services 打开了通道，无需再通过额外的技术绕行方案。

亚马逊总裁兼 CEO Andy Jassy 表示，「OpenAI 的这项公告很值得关注。我们很高兴在接下来的几周内，把 OpenAI 的模型直接引入到 Bedrock，供客户使用，同时也会配合即将推出的 Stateful Runtime Environment。这样一来，开发者在选择模型时会有更大的灵活性，可以根据不同任务挑选最合适的方案。」

以下为 OpenAI 与微软双方的联合公告内容：

在快速变化的技术环境下，OpenAI 和微软需要不断调整合作方式，以更好地服务客户并推动各自发展。今天，双方宣布对合作协议进行一次更新，目的是让合作关系更加清晰、灵活，同时提升确定性，并更好地把 AI 的价值带给更广泛的用户。

新的协议让双方在大规模建设和运营 AI 平台时拥有更强的可预期性，同时也保留了各自探索新机会的空间。

新协议主要内容包括如下：

一，微软仍然是 OpenAI 的主要云合作伙伴，OpenAI 的产品会优先在 Azure 上发布，除非微软无法或选择不支持相关能力。同时，OpenAI 现在也可以在其他云平台向客户提供产品。

二，微软对 OpenAI 模型和产品的知识产权许可将持续到 2032 年，但不再是独占授权。

三、微软不再向 OpenAI 支付收入分成。

四、OpenAI 向微软的收入分成将持续到 2030 年，比例保持不变，但设有总额上限，且不受技术进展影响。

五、微软仍然作为重要股东，继续直接参与 OpenAI 的成长。

虽然这次调整让合作关系更加简化，但双方的合作目标依然雄心勃勃：从扩展数据中心算力规模、共同推进下一代芯片研发，到在网络安全等领域应用 AI，双方将继续合作，把 AI 推向更大规模的实际应用。

有人发现，OpenAI 在与微软的新协议中取消了关于 AGI 的相关条款。考虑到其即将推进 IPO，有观点认为，未来正式对 AGI 作出官方界定或表述的可能性正在上升。

不再独家，但仍深度绑定

2019 年，微软向 OpenAI 投资约 10 亿美元，成为其关键战略合作伙伴。与此同时，OpenAI 完成从非营利向「有限盈利」结构的转型。

接下来的两年，这段合作显得低调却关键。GPT-3 等模型在 Azure 上完成训练，微软则持续为 OpenAI 构建专用 AI 超算系统。

2023 年，微软进一步加码投资（外界普遍认为规模达百亿美元级别），双方进入最紧密的合作阶段。一系列标志性动作接连发生：

GPT-4 优先在 Azure 上发布

Bing 引入 AI，对抗 Google 搜索

Microsoft 365 推出 Copilot

Azure OpenAI Service 成为开发者入口

2024 年以来，随着生成式 AI 竞争加剧，双方开始出现微妙变化。一方面，OpenAI 探索更多商业路径；另一方面，微软也逐步引入更多模型来源，降低单一依赖。

直到现在，双方重新划定合作边界，独家时代松动。
