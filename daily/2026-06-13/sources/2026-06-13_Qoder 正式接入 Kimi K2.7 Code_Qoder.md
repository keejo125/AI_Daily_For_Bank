# Qoder 正式接入 Kimi K2.7 Code

> 原文链接：https://mp.weixin.qq.com/s/oU6PFCfy2NuRO-kcwE9HkQ
> 公众号：Qoder

1 images

Qoder 正式接入 Kimi 最新的 K2.7 Code 模型，它不是简单的能力叠加，而是一次面向「长上下文编程」场景的专项进化。

和上一代 K2.6 相比，K2.7 Code 做了三件关键的事：

第一，指令遵循更精准。在长上下文窗口内，模型对复杂指令的理解和执行精度显著提升——不再「越长越漂」。

第二，长程任务更稳定。面对跨文件、跨模块的大型编程任务，K2.7 Code 的执行连贯性和正确率明显提高。

第三，思考更克制。大幅改善过度思考倾向，平均 token 消耗减少 30%——同样的任务，更快完成，更少冗余。

这三项提升直接转化为 Agentic 能力的增强：Qoder 的 Agent 模式在处理「理解 → 规划 → 执行 → 验证」全链路时，底层推理更高效、决策更果断。

目前 Qoder 国际版 Desktop、JetBrains 插件、CLI、QoderWake 已全面上线 Kimi K2.7 Code，6月15日 将开启 Fast 模式，享 6 倍速体验。

体验入口：https://qoder.com/
