# QoderWork新能力：帮你操作电脑桌面应用

> 原文链接：https://mp.weixin.qq.com/s/2NLYHuqFxYWfd-Wd0MWd1A
> 公众号：Qoder

有些活，AI 一直帮不上忙

这两年 AI 助手进步很快。接上各种工具，它能帮你查数据、发消息、建日程、写文档。只要应用开放了接口，AI 就能接管。

但总有一类活它碰都碰不了。

公司那套用了多年的老系统，没有接口；某个第三方平台的后台，只能在网页上一项项点；本地装的专业软件，只认鼠标和键盘……这些应用没有 API，没有连接器，没有命令行。AI 能想到该怎么做，却够不着你的屏幕。

于是这部分活又回到你手上：对着界面一个个点，一栏栏填，一遍遍切窗口。

现在，QoderWork 把这道坎也迈过去了。它能像人一样看着屏幕、动手操作，那些没有接口、只能点鼠标的活，从此也能交给它。

QoderWork Computer Use 功能上线。

那些原来只能点鼠标的活，现在交给它

开启「计算机控制」后，QoderWork 可以在你授权的应用里执行操作：点击、输入文字、滚动页面、按快捷键、拖拽元素、操作菜单。基本上你用鼠标键盘能做的事，它都能做。

举个例子。周五下午你要提交本周出差的报销单。以前的流程：打开 OA → 找到报销入口 → 新建单据 → 选费用类型 → 填金额 → 选日期 → 上传发票 → 填备注 → 选审批人 → 提交。大概要点击十几次、切换三四个下拉菜单。

现在你可以这样说："帮我打开 OA 报销页面，新建差旅报销。机票 3200，5 月 26 日；酒店两晚共 1600，5 月 26-27；市内交通 340。审批人选张经理，提交。"

QoderWork 打开浏览器或应用，找到对应表单，逐项填入，该选下拉框选下拉框，该点日期选日期，最后提交。你全程不用碰鼠标。

它怎么做到准确操作的？

这里有一个关键的技术差异。蒙着眼睛摸电梯按钮，靠的是记忆和手感，按错了只能再试。但如果你能看到每个按钮上的编号，就不会按错。

市面上多数方案走第一种路线：截一张屏幕的图，让模型猜坐标。QoderWork 走的是第二种：它直接读取界面的结构化信息，每个按钮、输入框、菜单项都有明确编号，操作时按编号精准定位。不存在"大概是这个"的模糊空间。

而且是闭环的，操作一步，观察结果，再决定下一步。遇到弹窗、加载延迟、页面跳转，自己判断应对，不是死板地按脚本跑。

更多可以交给它的事：

"把这张 Excel 里的几十条数据，逐条录进公司的业务系统。" ——内部老系统不支持批量导入，平时只能一条条手敲。

"登录平台后台，把这批条目按表格逐项更新一遍。" ——后台没有批量接口，挨个点开太费时间。

"把老系统里上个月的记录逐笔导出来，汇总成一张表。" ——系统没有导出功能，全靠人工复制粘贴。

规律就一句话：凡是"你知道该怎么做，但做起来全是鼠标点击"的活，都可以交给它。

关于安全和可控

"AI 能操作我的电脑"这句话听起来让人紧张。所以这里需要讲清楚几件事。

你始终握着开关。计算机控制是一个你随时可以关掉的连接器，不是后台默默运行的东西。关掉开关，QoderWork 立刻失去操作桌面的能力，回到纯对话模式。

桌面操作需要你主动开启。计算机控制默认关闭，需要你在设置里手动打开，并且完成系统权限授权后才能工作。

简单说：默认关闭，开关在你手上，授权之后才动手。

开启方式

升级到最新版，打开 QoderWork → 扩展 → 连接器 → 已安装，在「市场」区域找到「计算机控制」，打开开关。首次启用会请求辅助功能和屏幕录制两项系统权限，按提示授权即可。

不需要记任何特殊指令。像平时一样跟 QoderWork 说话，当任务需要操作桌面时，它自动调用对应能力完成。

计算机控制已在最新版 QoderWork 中上线。开启它，把那些"全是鼠标点击"的活交出去，自己腾出手做更要紧的事。
