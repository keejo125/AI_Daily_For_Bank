# 马斯克变身「算力包租公」！砸数万GPU疯狂喂养Cursor，联手反杀OpenAI

> 原文链接：https://mp.weixin.qq.com/s/meJ2zWk8Czk5Vu-OVmVM6A
> 公众号：新智元

新智元报道

编辑：犀牛

【新智元导读】

当OpenAI 还在抢模型话语权时，马斯克已经一手把 xAI 变成「算力包租公」，一手 Tesla AI5、AI6 与 Dojo 芯片——这个男人的AI帝国，永远比你想象的更疯狂。

万万没想到，2026年最会玩商业模式的科技大佬，竟然是那个在火星上画大饼、在地球上狂建数据中心的钢铁侠。

就在所有人都以为马斯克的xAI闷头训练Grok、跟OpenAI死磕的时候，这哥们悄悄干了件让整个硅谷都傻眼的事——

把自己的算力拿出来租了

。

没错，那个坐拥超过55万张GPU、号称要在五年内拥有比全世界加起来还多算力的男人，现在准备当「包租公」了。

而他的第一个大客户，是AI编程领域的当红炸子鸡——Cursor。

你训模型，我收租金

据知情人士透露，Cursor正在使用xAI的基础设施来训练其最新的AI编程模型Composer 2.5。

具体有多少算力？

数万张

GPU

。

放在两年前，整个OpenAI训练GPT-4也就用了2.5万张A100。

现在马斯克随手一租，就是这个量级。

更骚的是，这笔交易的本质——

xAI从一个纯粹的AI模型公司，摇身一变成了云服务提供商。

亚马逊AWS、微软Azure、谷歌云多年苦心经营的商业模式，马斯克用一个数据中心就给复制了。

但这招绝不只是为了赚点租金。

想想看，Cursor手里有什么？

海量的开发者编程数据

。

全球数百万程序员每天在Cursor上写代码、改bug、做重构，这些数据对于训练编程AI来说，简直是黄金矿脉。

所以这笔交易的真正逻辑是：

你拿我的GPU训模型，我顺便看看你的数据有多香。

双方各取所需，OpenAI看了直呼内行。

Colossus：全球最疯狂的GPU集群

xAI的数据中心项目「Colossus」（巨像），光听名字就知道不是善茬。

这玩意儿建在田纳西州孟菲斯，目前已经扩展到

接近2吉瓦

的算力规模——差不多相当于一座中型核电站的发电量。

2024年，xAI用

122天

建成了第一个拥有10万张H100 GPU的超级集群。

英伟达CEO黄仁勋看了直摇头：

这活儿正常得干4年。

到2025年底，Colossus的GPU数量已经超过了55万张，包括H100、H200和最新的Blackwell GB200。

马斯克甚至在圣诞节那几天宣布买下第三栋建筑，并起了个让微软想打人的名字——「MACROHARDRR」（比「微软」更硬）。

按照马斯克的规划，xAI将在五年内拥有5000万张「H100等效」GPU。

这个数字意味着整个OpenAI、Anthropic、Google、Meta加起来，可能都打不过他。

但拥有最多的GPU，不代表用得最好。

这就是xAI当前最尴尬的问题。

挖角Cursor高管：马斯克的「抢人战术」

xAI与Cursor称得上是相爱相杀。

3月底，xAI最后两位联合创始人Manuel Kroiss和Ross Nordeen打包走人。

至此，xAI创始团队

全军覆没

。

创始人跑了怎么办？

马斯克的答案简单粗暴：

抢别人的

。

抢的就是Cursor。

3月12日，xAI宣布挖来了Cursor的两位高管——Andrew Milich和Jason Ginsberg。

这两人之前共同领导Cursor的产品工程团队，直接把Cursor做到了20亿美元年化收入、293亿美元估值。

现在，他们将直接向马斯克汇报，负责重建xAI一直做不好的AI编程工具「Macrohard」。

xAI收购了Cursor的人才，又租给Cursor算力。

这算是什么关系？塑料兄弟情？还是商业互利？

反正OpenAI看了估计得流泪：我培养的人才被挖，我的竞争对手还在用别人的GPU变强。

AI圈的生态，真是越来越魔幻了。

特斯拉AI5芯片：40倍性能提升

如果说xAI的故事是「乱中求稳」，那特斯拉的芯片战略就是「稳中求进」。

4月15日凌晨，马斯克在推特上晒出了一张照片：

特斯拉AI5处理器的首个样品

。

这颗芯片的参数相当炸裂：

体积

：仅占半个光罩大小（约400平方毫米）

性能

：在某些指标上比AI4提升

40倍

内存

：是上一代的9倍

计算能力

：是上一代的8倍

功耗

：目标250瓦

AI5采用了

12颗

SK

海力士内存

（可能是GDDR6/7），384位内存接口，带宽高达768GB/s到1.5TB/s。

马斯克发推感谢「TSC」和三星的支持，结果网友发现他把台积电（TSMC）的官方账号搞错了——@了一家同名的台湾半导体公司。

这波操作，真的很马斯克。

AI5为什么如此重要？

因为它关系到特斯拉的三大核心业务：

全自动驾驶（FSD）、Cybercab无人出租车、以及Optimus人形机器人。

马斯克在2025年Q3财报电话会上亲口说过：解决AI5的问题，对特斯拉来说是

存在级

的。

他甚至连续几个月每周六亲自跟芯片团队开会。

现在AI5流片成功，意味着最难的设计阶段已经过去。

接下来就是送到TSMC和三星代工厂量产，预计2027年中开始大规模部署。

根据马斯克的透露，AI6芯片将具有以下特点：

制程

：三星2纳米（得克萨斯工厂），后续版本AI6.5将使用TSMC亚利桑那的2纳米工艺

内存

：LPDDR6，解决AI5因赶进度而做出的设计妥协

性能

：在相同面积下，实现比AI5

真正翻倍

的性能提升

芯片面积

：依然保持半个光罩大小

发布时间

：预计2027年量产

马斯克特别强调了一点：AI5和AI6都有大约一半的TRIP AI计算加速器专门用于SRAM缓存。

这意味着

有效内存带宽比DRAM带宽高一个数量级

——

对于那些需要频繁访问内存的AI计算来说，这是巨大的性能提升。

更有意思的是，AI6的主要优化对象是Optimus机器人和Robotaxi。

马斯克的算盘打得很清楚：

未来特斯拉卖的不是汽车，而是会移动的AI计算平台。

Dojo 3复活

但真正让业内惊掉下巴的，是另一条消息：

Dojo 3回来了

。

背景是这样的：特斯拉的Dojo超级计算机项目自2023年启动以来，一直是马斯克的心头好。

它采用了独特的「晶圆级」架构，把整片晶圆做成一颗巨型芯片，理论上可以实现更低的通信延迟和更高的训练效率。

但2024年8月，消息传出Dojo项目被「无限期搁置」，团队解散，项目负责人Peter Bannon退休。

当时外界普遍认为，Dojo已经「黄了」。

然而就在1月19日，马斯克突然宣布：AI5芯片设计状态良好，特斯拉将重启Dojo 3的工作。

这个反转来得太快，让所有分析师都措手不及。

按照马斯克的说法，Dojo之所以被暂停，是因为特斯拉必须集中资源解决AI5的问题。

现在AI5搞定了，团队可以重新分配精力攻克Dojo 3。

更关键的是，AI6和Dojo 3可能会采用

融合架构

——同一套指令集、同一套软件栈。

这意味着特斯拉可以用同一颗芯片，既装在汽车和机器人里做推理，又堆在数据中心里做训练。

这个愿景如果实现，

特斯拉将成为唯一一家从芯片到整车到机器人到云端全部自研的公司。

英伟达看了沉默，英特尔看了流泪。

马斯克的AI帝国

正在重塑游戏规则

回顾整个故事，马斯克正在用他擅长的「硬件思维」重塑AI竞争格局。

当OpenAI、Anthropic、谷歌们还在绞尽脑汁优化算法的时候，马斯克直接从源头下手：

自己建数据中心、自己设计芯片、自己搞超级计算机。

他的逻辑很简单：AI竞争的终局，不是谁的算法最聪明，而是谁的算力最充沛、谁的成本最低。

就像当年他用猎鹰9号火箭把NASA和波音卷到哭一样，他现在准备用AI5和Dojo把英伟达和AWS卷到哭。

当然，风险也是显而易见的。

xAI的创始人跑光了，GPU利用率只有11%，Grok的产品问题不断。

但如果说有一个人能在一片混乱中杀出一条血路，那大概就是这个在推特上把台积电名字都能搞错的男人。

毕竟，他连火星都敢去，还怕什么。
