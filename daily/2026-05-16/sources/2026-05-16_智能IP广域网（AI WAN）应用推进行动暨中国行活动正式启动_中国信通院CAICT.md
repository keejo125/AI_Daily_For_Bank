# 智能IP广域网（AI WAN）应用推进行动暨中国行活动正式启动

> 原文链接：https://mp.weixin.qq.com/s/U5JAmuDKgKEEi2zC57vpWw
> 公众号：中国信通院CAICT

2026年5月16日，2026世界电信和信息社会日大会在武汉举行。在“AI智算网络与应用专题研讨会”上，中国人工智能产业发展联盟（AIIA）信息通信业AI创新推进委员会、中国信息通信研究院（简称“中国信通院”）联合中国通信标准化协会、中国电信、中国移动、中国联通等产业伙伴，共同发起“智能IP广域网（AI WAN）应用推进”行动暨中国行活动。

行动将基于基础电信企业已有的创新实践，聚焦四大领域：一是以弹性、高效的企业入算网络赋能行业企业低成本、高效率利用算力资源；二是以灵活、快速的边缘入算网络推动智慧家庭各类算力服务创新；三是以高速、无损的新型城域网络支撑分布式训练、推理等人工智能新型业态；四是以安全、可靠的高速数据网络支持数据要素高效流通和管理。

AIIA信息通信业AI创新推进委员会秘书长、中国信通院技术与标准研究所所长张海懿表示，通过发起此次行动，我们期望协同产业各方力量，加快推动智能 IP 广域网（AI WAN）应用，推进算网融合创新服务模式成熟，强化网络对产业智能化发展的赋能作用。

我国持续推进 “人工智能 +” 行动，推动行业人工智能规模化落地。工业和信息化部、国家发展改革委等部委陆续发布了《算力基础设施高质量发展行动计划》、 “东数西算”公共传输通道等相关行动，智能 IP 广域网（AI WAN）对人工智能产业发展的核心支撑价值愈发凸显。

当前，智能IP广域网已在中国电信、中国移动、中国联通三大运营商规模部署，有力支撑训练推理、数据流通、智能服务发展，广泛赋能交通、医疗、制造、科研等行业应用，有效推动网络与业务向更智能、更高效、更安全的方向演进。

本次AI WAN中国行活动立足行业需求，将走进全国重点城市与重点行业，广泛开展调研，深化案例分析，凝聚产业共识，为人工智能和数字经济高质量发展注入新的动能。
