# 龙虾之父月烧940万元的token！要不是入职OpenAI还真用不起

> 原文链接：https://mp.weixin.qq.com/s/mYnv6TEwGK4VsXurvH3URg
> 公众号：量子位

衡宇 发自 凹非寺

量子位 | 公众号 QbitAI

龙虾之父Peter Steinberger

，今天在𝕏上晒出了一张自己的CodexBar后台截图。

一张相当离谱的截图——

上面透露出的信息和数字让我眼睛都瞪大了：

过去30天，他调用的OpenAI API总费用达到1305088美元

，约合人民币940万元；

同时消耗6030亿token，发起760万次请求；

最常用的模型是GPT-5.5。

以上所有费用，由OpenAI承担。

看呆的人不止是我，不少网友也发出惊呼：

Bro，你最好拿出点儿价值百万美元的工程师都做不到的东西，不然这可能就是前沿实验室泡沫破裂的开端了。

而且你享受的还是补贴价格，我的天。

如果是其他人按实际成本调用这老些token量，价格？不敢想不敢想。

大头都花在开发OpenClaw身上了

这些数都数不清的token，虾爹基本都用来开发OpenClaw了。

他表示，被OpenAI买下后，自己的团队现在只有3个人。

这个时代，团队人数贵精不贵多，更何况虾爹团队还有庞大的硅基外援！

虾爹说，他们会

同时在云端运行约100个Codex

。

这是虾爹的英明之举——没有选择做“一个超级万能Agent”，而是拆成大量小Agent。

这其实也是近期很热门的话题之一，即“多Agent群体协作”。

这些Codex会自动Review PR、扫描安全漏洞、查重Issue、编写修复程序、监控Benchmark回归，甚至还能监听会议内容，然后自己创建PR。

而且是彼此之间互相协作那种，是互相Review、互相监督、互相补漏洞。

虾爹本人也很坦诚啦，

明说虽然团队开发完全使用Codex编写，但“我修复的一些比较混乱的PR很可能是Claude做的”。

（注：该团队还使用Clawpatch.ai、Vercel的Deepsec和Codex Security进行漏洞和安全分析。）

这100个Codex里面，一些Agent负责干活，另一些Agent负责盯着其它Agent干活……

讲道理，这情况乍一看清奇，仔细想想又十分合理。

人类负责目标，Agent负责执行，整个软件开发流程开始出现一种非常明显的“流水线化Agent协作”。

这画风，不就像一家真正的软件公司吗？？？

只不过员工从人类逐渐变成了AI Agent，苦笑摊手.jpg。

此外，仔细看虾爹放出的Codexbar可以发现，他的每日请求量达到20.6万次。

折算下来，

每秒大概调用2.4次。

这个信息其实比虾爹在用Codex养蛊

（不是）

更有意义，因为这意味着所有的Codex都在长时间、持续地稳定干活。

众所周知，Agent一旦长时间在线运行，随随便便就会被很多实际场景里的难题噎住，比如上下文信息杂乱冗余、记忆数据不断臃肿balabalabala，又或者多个Agent之间互相干扰误导之类的。

一个月，要！花！好！多！钱！啊！

面对130万美元/30天的天价账单，虾爹回应了质疑。

虽然每个月的token调用量让人望洋兴叹，但这只是虾爹开了快速模式的结果。

为了让这些Agent持续高频协作、快速响应，系统需要维持非常激进的推理调度，这会直接推高token消耗。

我可以关闭快速模式，这样成本会降低70%。

允许Agent“慢一点”，成本立刻就能下来。听起来确实节约了非常多。

But，网友一拨算盘立刻就发现，

即使节约了70%的成本，虾爹的每月token费用还是有40万美元之多。

也就是272.396万人民币。

这，这

只是虾爹一个人过去30天的token调用费用

啊！

你知道吗，今年3月的时候，路透社消息就称，OpenAI计划将员工数量拓展到8000人……

回过头来，虾爹讲了“这压根不算贵”，因为他的思路完全是另一套逻辑

当被问及投资回报率时，他表示，他的团队开发的所有产品都是开源的，并且兼容主流模型和开源模型，

“回报率相当高。”

他还说：

禁用快速模式后，我的花费比一个工程师的费用还低，而且效果绝对好得多。

So，虾爹压根不是拿Codex和“工具成本”比，而是拿它和“工程团队成本”比，尤其是在旧金山。

一个高级工程师一年几十万美元，也不算什么稀罕事。

而OpenClaw现在让100个Agent承担大量重复工程劳动，这些劳动内容非常机械化，AI天然适合被用来替代原本需要的大量工程人力。

奥特曼此前在公开发言中提到，未来会是一个“extremely multi-agent”的世界。

现在虾爹治下的OpenClaw，已经开始有那个味道了。

按照这个趋势下去，软件开发将从“人类使用工具”，慢慢变成“人类管理Agent团队”。

换句话说，“月烧130万美元”是便宜还是贵，完全取决于你的视角。

One More Thing

既然虾爹说巨量token大部分都花在开发OpenClaw身上了，那剩下的呢？

虾爹也不遮着藏着，表示——

“我同时还在筹备几个创业项目哦～”

虾爹显然已经不满足于只养100个Agent来搞OpenClaw的开发的。

至于新的创业项目们到底是什么，大家期待一波吧！
