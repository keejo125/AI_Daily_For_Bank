# 谷歌扩大 SynthID 在 AI 水印技术中的应用，并提供内容检测 API 预览

> 原文链接：https://mp.weixin.qq.com/s/bMNlkT6a4bkY3SnfrQNx_Q
> 公众号：InfoQ

作者 | Sergio De Simone

译者 | 平川

谷歌的 SynthID 旨在将难以察觉的信号嵌入 AI 生成的内容中。现在，在获得英伟达（Nvidia）和 OpenAI 等多家行业领军企业的采用后，他们正 在谷歌云的 Gemini 企业代理平台上新增一项内容检测 API。

谷歌正向包括 Shutterstock、Snap、Fox Sport、Canva 等在内的一批可信赖的合作伙伴提供该 API 的预览版，以便根据他们的反馈进行优化。该 内容检测 API：

为企业提供了一种强大的工具，使他们能够识别由谷歌及其他流行的模型所生成的 AI 内容，从而帮助企业决定如何评估和管理其平台上的媒体内容——无论是用于内容排序和防范保险欺诈等后台运营，还是用于事实核查和标注合成媒体等面向用户的内容。

该 API 通过 REST 接口接受 JPEG、PNG 或 WebP 格式的图像，利用机器学习模型分析像素级伪影、噪声模式和光谱异常，并将分析结果包含在响应中。据谷歌称，该 API 不会存储或保留经它处理的图像。

谷歌还指出，近几个月来， SynthID 的应用范围已经扩展至英伟达（Nvidia）、OpenAI、Kakao 和 ElevenLabs。英伟达将在其 Cosmos 基础模型 中使用 SynthID。OpenAI 将采用多层方案，结合 C2PA 元数据和 SynthID 水印技术，增强验证过程的抗干扰能力：

这两个系统相辅相成。C2PA 有助于在内容中包含详细的上下文信息；SynthID 则有助于在元数据丢失时保留信号。水印技术能够通过截图等转换操作变得更为持久，而与单纯使用水印相比，元数据能提供更丰富的信息。

有意提前使用该 API 的组织可以 向谷歌申请。

此外，谷歌最近 在其 Gemini 应用中新增了基于 SynthDI 的音频、视频和图像验证功能，并计划将其扩展至搜索和 Chrome 浏览器。此外，该公司还将把 C2PA 内容凭证的应用范围扩展至 Pixel 手机，包括 Pixel 8、9 和 10 ，为使用相机应用拍摄的图像和视频附加元数据，以便证明这些内容并非由 AI 生成。

SynthID 是一款用于给 AI 生成的图像添加水印并进行识别的工具，它将数字水印直接嵌入到图像像素中，肉眼无法察觉，但可以通过检测进行识别。自 三年前推出 以来，谷歌已经将 SynthID 集成到其生成式媒体模型和产品中，并利用该技术为超过 1000 亿张图片和视频以及时长 6 万年的音频内容添加了水印。
