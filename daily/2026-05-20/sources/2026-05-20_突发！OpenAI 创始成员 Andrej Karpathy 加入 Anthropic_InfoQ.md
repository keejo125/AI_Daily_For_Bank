# 突发！OpenAI 创始成员 Andrej Karpathy 加入 Anthropic

> 原文链接：https://mp.weixin.qq.com/s/4m9rJqBVC-ZSyt_KRxfuXg
> 公众号：InfoQ

编辑｜四月

刚刚，Andrej Karpathy 在 X 上发布个人动态，称已经加入 Anthropic。

他表示，未来几年将是大语言模型前沿发展尤其关键的阶段，自己很高兴加入 Anthropic，并重新回到研发一线。同时，他也提到，教育仍然是自己长期热爱的方向，后续会在合适时间恢复相关工作。

这件事值得关注，并不止于一位知名开发者加入了头部AI公司，更加在于 Karpathy的履历实在太特殊。

他是 OpenAI 创始成员之一，曾担任特斯拉 AI 和 Autopilot 相关负责人，也是斯坦福深度学习课程 CS231n 的核心推动者之一。对很多 AI 从业者来说，Karpathy 不只是研究员，更是AI 圈里的大神。

从 nanoGPT、Neural Networks: Zero to Hero，到后来围绕 LLM 和 AI 编程的一系列长文，

他长期扮演着技术布道者和开发者心智塑造者的角色。

2024 年离开 OpenAI 后，Karpathy 创办了 AI 教育平台 Eureka Labs，试图把 AI 和教育结合起来，打造更原生的 AI 学习方式。但这次加入 Anthropic，意味着他阶段性地把重心重新转向了前沿研发。更准确地说，这不是简单“暂停教育”，而是一个信号：当 LLM 前沿窗口再次变得足够关键，Karpathy 选择回到牌桌上。

值得注意的是，Claude Code 的创建者、负责人 Boris Cherny 已经在评论区第一时间表示欢迎。Claude Code 也是 Anthropic 近一年最受开发者关注的产品线之一，从最初的终端原型，逐步变成面向软件工程和专业工作的 Agentic Coding 工具。

不过，目前 Karpathy 并没有公开说明自己将加入 Anthropic 的哪个具体团队，所以暂时不能直接判断他会进入 Claude Code 团队。但从他的近期关注来看，外界把这次加入和 AI Coding、Agent、科研教育联系起来，并不意外。

前段时间，Karpathy 开源的 autoresearch 项目已经指向了一个更激进的方向：

让 AI Agent 自己修改训练代码、运行实验、根据指标保留或丢弃改动，并持续循环

。换句话说，它不只是让 AI 写代码，而是尝试让 AI 进入“提出假设、跑实验、看结果、继续迭代”的研究流程。

所以，这次 Karpathy 加入 Anthropic，真正值得看的不只是“大神跳槽”本身，更是它背后指向的方向感。

最后，也希望奥特曼能够尽早对此释怀～～
