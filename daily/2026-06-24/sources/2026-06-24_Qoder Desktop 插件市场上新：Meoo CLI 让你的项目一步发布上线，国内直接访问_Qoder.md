# Qoder Desktop 插件市场上新：Meoo CLI 让你的项目一步发布上线，国内直接访问

> 原文链接：https://mp.weixin.qq.com/s/VaAxFQq3V0Jp9igz8ND3-g
> 公众号：Qoder

Meoo CLI 正式上线 Qoder 插件市场。

现在，你可以在 Qoder Desktop 中安装 Meoo CLI skill，通过自然语言为项目接入数据库、用户登录、文件存储、AI 模型等云端能力，并将本地项目发布为国内可直接访问的线上应用。

Qoder Desktop 负责把想法快速做出来，Meoo CLI 负责让它跑在线上。

什么是 Meoo CLI？

Meoo CLI 是阿里秒悟推出的全栈应用构建命令行工具，覆盖从项目初始化到上线发布的完整流程。

当你有了一个想法，或者已经完成了本地项目，可以用自然语言告诉 Qoder：

使用 Meoo CLI，把这个项目发布到线上。

Qoder 会理解当前项目结构，通过 Meoo CLI 完成云服务开通、数据库配置、服务端部署和 CDN 发布。你不需要离开 Qoder，不需要分别配置多个云平台，就能把项目推进到线上可用状态。

Meoo CLI 帮你的项目补齐什么？

Meoo CLI 覆盖了一个应用从本地到线上所需的完整能力：

数据库：

项目需要保存用户数据时，直接开通并接入云端数据库

文件存储：

项目需要上传图片或文件时，接入云端对象存储

用户登录：

项目需要区分用户身份时，接入认证服务

AI 模型：

项目需要 AI 识图、文生图或语言模型时，调用秒悟内置模型服务

服务端逻辑：

项目需要后端接口时，部署边缘函数处理业务逻辑

CDN 发布：

项目完成后，一键发布到国内 CDN，生成可直接访问的线上链接

国内直接访问，不再依赖 Vercel

以前，很多人用 Vercel 部署项目。但 Vercel 在国内需要科学上网才能访问。你部署完分享给朋友、同事或客户，对方很可能根本打不开。

Meoo CLI 发布的项目托管在国内节点，访问者无需任何代理工具，打开链接即用。对于面向国内用户的项目，这意味着你做出来的东西可以真正被别人看到、使用和传播。

实战案例：世界杯本命球队测试器

世界杯正在进行。我们用 Qoder Desktop + Meoo CLI 做了一个互动小工具：

世界杯本命球队测试器。

用户上传一张照片，AI 从表情、色彩和风格维度进行分析，匹配世界杯热门球队中最适合用户气场的一支，生成专属分析结果和分享链接。

这个项目用到了 Meoo CLI 的三项核心能力：

AI 模型：

调用识图能力分析用户照片

数据库：

保存每次测试结果，支持"签约档案"历史记录查看

云端发布：

发布为国内可直接访问的线上应用，用户打开即玩

从一个本地 Demo 到一个可在线参与、生成结果、保存记录并分享传播的世界杯互动应用，全程在 Qoder Desktop 中完成。

如何在 Qoder Desktop 中使用 Meoo CLI？

打开 Qoder Desktop 的 Quest 视窗，进入左侧「插件市场」，找到 Meoo，点击「立即试用」即可。

安装完成后，试试对 Qoder Desktop 发送：

用 Meoo CLI 把这个项目发布上线。

或者直接通过 /meoo-cli 唤起直接调用。

把你的本地项目，变成可以访问、使用和分享的线上应用。
