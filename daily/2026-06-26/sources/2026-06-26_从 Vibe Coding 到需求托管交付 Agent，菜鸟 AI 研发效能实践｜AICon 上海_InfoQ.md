# 从 Vibe Coding 到需求托管交付 Agent，菜鸟 AI 研发效能实践｜AICon 上海

> 原文链接：https://mp.weixin.qq.com/s/204-dpWU5IytY081CNJtUA
> 公众号：InfoQ

当前，以 Agent 为核心的新一轮技术浪潮正在席卷产业。那么，世界模型的下一个突破在哪？Agent 从 Demo 到工程化还差什么？研发体系不重构，还能撑多久？

6 月 26 日 -6 月 27 日，AICon 全球人工智能开发与应用大会 将在上海举办。本次大会将围绕以上问题，邀请来自腾讯、阿里、快手、华为、飞猪等 50+ 头部企业的技术负责人、高校与科研机构的一线专家，分享 Agent 在真实生产环境中的落地经验与前瞻思考。深入探讨 Agent 从原型到量产的工程挑战、数据与记忆的基础设施底座、安全可信的落地保障，以及大模型推理优化、智算架构升级等关键命题。

菜鸟网络研发总监郭凤钊已确认出席 “企业级研发体系的重构” 专题，并发表题为《从 Vibe Coding 到需求托管交付 Agent，菜鸟 AI 研发效能实践》的主题分享。AI Coding 工具正在快速渗透研发流程，但大部分企业的落地仍停留在“辅助写代码”阶段，缺乏体系化的工程方法论。菜鸟 AI 研发效能团队从 2023 年开始推进 AI Coding 落地，AI 代码贡献率从 10% 提升至 80%+，部分团队接近 100%。经历了从 Vibe Coding 到 Harness Engineering 的演进，本次分享将完整呈现这条从工具采用到工作流重构的实践路径，以及过程中踩过的坑。

郭凤钊，菜鸟 AI 研发效能负责人 / 架构师，负责研发效能、稳定性与 FinOps 体系，双 11 等大促技术 PM。专注于 AI 赋能企业研发效能的实践与落地，是菜鸟内部 AI 技术布道的核心推动者。鲲鹏会杭州会员。他在本次会议的详细演讲内容如下：

演讲提纲：

从 Vibe Coding 到 Harness Engineering，近 3 年的认知升级路径

以 Agent-Friendly 的理念重塑需求交付全流程

硅基程序员，实现从 Coding Copilot 到需求的托管交付

研发效能度量的世纪难题与 AI Coding 的价值呈现

听众收益：

拿到一套可迁移的 AI Coding 工程化方法论。 分层上下文管理、Playbook 驱动的流程编排、确定性质量门控，可以直接迁移到自己团队的技术栈上

理解从 Vibe Coding 到生产级 AI 工程的完整演进路径和踩坑经验。 包括微服务场景的失败模式、上下文污染的解决方案、度量指标的坑

获得"硅基程序员"从概念到落地的具体架构参考。 完整呈现一个 AI Agent 如何从 Copilot 级别的辅助工具演进为可托管需求交付的硅基员工

除此之外，本次大会还策划了 端侧 AI、物理与数字空间智能化、世界模型与多模态智能突破、Agent 架构与工程化实践、Agent 安全与可信治理、企业级研发体系重构、AI 原生数据工程、AI 时代的个人提效与组织变革 等 14 个专题论坛，届时将有来自不同行业、不同领域、不同企业的 50+ 资深专家在现场带来前沿技术洞察和一线实践经验。
