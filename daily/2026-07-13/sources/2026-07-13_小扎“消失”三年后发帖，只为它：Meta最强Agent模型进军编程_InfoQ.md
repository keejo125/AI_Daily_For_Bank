# 小扎“消失”三年后发帖，只为它：Meta最强Agent模型进军编程

> 原文链接：https://mp.weixin.qq.com/s/mJF_5Sc4u6C7cMKGqAwpRQ
> 公众号：InfoQ

整理 | 华卫

在 AI 负责人 Alexandr Wang 领导下推出首款人工智能模型三个月后，Meta 再次推出一次重大升级，试图在 AI 市场的关键领域与 OpenAI 和 Anthropic 展开竞争。

昨日，Meta 正式发布了新版 Muse Spark，这是一款面向智能体编程（agentic coding）的多模态 AI 模型，目标是与 OpenAI 和 Anthropic 等公司的同类产品展开竞争。Wang 在接受采访时表示，Muse Spark 1.1 是“目前在智能体任务和编程领域能力最强的模型”。

过去几年里，Meta 已经陆续推出了多款基础 AI 模型。而此次 Muse Spark 的发布显然足够重要，甚至促使 CEO Mark Zuckerberg 时隔三年首次在 X 平台发帖。上一次 Zuckerberg 发帖是在 2023 年 7 月，当时该平台刚从 Twitter 更名为 X。

他在此次帖子中称，Spark 是“一款价格极低但能力强大的智能体和编程模型”，并指出该模型“最擅长智能体性能、工具调用以及计算机操作”。Zuckerberg 还透露，“很快还会有更多内容发布”，暗示 Meta 计划继续推出更多模型。

1

三个月性能显著飞跃，“智能体任务表现超过竞争对手”

Meta 向用户主打的卖点，是 Spark 处理大规模智能体任务的能力，包括修复代码漏洞、协助大型代码迁移等，这类自动化能力正是越来越多企业希望通过 AI 公司实现的方向。作为一个为智能任务打造的多模态推理模型，Muse Spark 1.1 在工具和计算机使用、编码以及多模态理解方面取得了重大提升。

在博客中，Meta 写道：“Muse Spark 1.1 在个人智能体任务中展现出卓越性能，尤其适用于需要跨多个外部应用和服务进行规划与协调的场景。”在多应用计算机使用流程中，Muse Spark 1.1 表现出色，通过在长时间会话中保持上下文，并在每个步骤智能选择脚本、直接界面交互和批量操作之间。它能够在极少的人为干预下导航陌生的界面。Wang 透露，在一些需要与各种第三方编程产品和服务进行交互的任务中，Muse Spark 1.1 的表现超过了竞争对手模型。

据其介绍，Wang 领导的 Meta 超级智能实验室（Meta Superintelligence Labs，MSL）针对编程相关任务对 Muse Spark 1.1 进行了训练，因为这最终能够提升 AI 智能体的整体能力，使其可以像“一群人类实习生”一样自主执行多项任务。“你必须把编程能力作为整体智能体能力的一部分来构建。”Wang 表示。

Wang 还分享道，自己一直在“内部试用”（dog-fooding）最新版本的 Muse Spark，并对其作为个人健康提升工具的潜力感到兴奋。例如，该模型可以帮助用户搜索网页、阅读学术论文，以及访问个人健康相关数据。谈到自己使用 AI 辅助健康管理的实验时，Wang 表示，“这是我认为真正体现智能体系统需求的一类应用场景。”

据了解，Muse Spark 最初的内部代号是 Avocado（牛油果），今年 4 月发布了初代版本。Meta 表示，首个于今年 4 月公布的 Spark 1.1 版本具备多步骤推理能力，可以处理复杂流程、管理数字化工作流，并能够在企业系统中部署新功能。相比初代模型，Muse Spark 1.1 在复杂功能实现、端到端开发任务以及代码库搜索与理解方面实现了显著的性能飞跃。

Meta 透露，当前 Muse Spark 1.1 被广泛应用于 Meta 的编码和研究工作流程，在 Meta 内部编码基准测试中与领先模型竞争。其研究人员现在通过在工作流程中利用 Muse Spark 1.1，自动化模型开发和评估任务。在感知和多模态推理方面，Muse Spark 1.1 也表现出色，能够检查视觉和音频输入，在漫长的工作流程中保留细节，并在真实执行环境中加以行动，它在视觉转代码生成、丰富的图像 / 视频字幕和智能计算机使用方面展现出特别优势。

2

进军 AI 编码市场，定价“非常激进且有吸引力“

在这一领域，Meta 目前相比竞争对手稍显落后。Anthropic 和 OpenAI 已经推出类似模型一段时间。但这并不意味着 Meta 的入场不会构成威胁。

AI 行业持续存在的一个核心竞争点，是模型的使用成本，而 Meta 此次似乎试图以价格优势切入市场。据外媒报道，Meta 将对 Spark 收取每百万输入 Token 1.25 美元、每百万输出 Token 4.25 美元的费用。这一价格高于 OpenAI 入门级模型 GPT-5 mini 以及 Anthropic 低成本模型 Claude Haiku 4.5，与 GPT-5.6 Luna 基本处于同一水平（略高一些），但低于 Anthropic 更高端的 Claude Sonnet 4.6。

Wang 表示，相比 Anthropic 和 OpenAI 等实验室提供的类似产品，Muse Spark 此次更新后的定价“非常激进且具有吸引力”。据悉，每个新的 API 账户都会获得 20 美元的免费额度用于测试模型，之后才转为按使用量付费模式。“我们的目标是真正提供具有吸引力的价格，并且能够随着大规模使用需求增长而扩展。”

并且，Wang 称，Meta 训练 Muse Spark 1.1 时，让它“能够很好地适配开发者目前使用的所有主流工具链”，而 Meta 认为这是实现模型最大规模普及的最佳方式。

“如果 Muse Spark 1.1 在编程能力上确实能够与 Claude 和 GPT 竞争，那么 Meta 可能终于找到了一个更清晰的商业化路径，将 AI 模型转化为付费开发者工具。”Futurum Equities 首席市场策略师 Shay Boloor 表示。

3

拟取代部分 Llama 模型，开源版正在开发中

此前，初代 Muse Spark 仅向“部分合作伙伴”开放，这些用户只能通过“私有 API 预览”方式访问该技术。目前，美国开发者已经可以通过 Meta Model API 公开预览访问 Muse Spark，用于测试提示词、比较模型输出以及开发原型集成方案。此次，Meta 通过开发者门户开放了新版模型 API 的公开预览，用户可以注册并查看集成指南。Meta 发言人表示，一些早期合作伙伴已经可以使用 API，而新用户“可以加入等待名单，之后会逐步获得访问权限”。

新版模型目前已经在 Meta AI 应用和网站的“Thinking 模式”中上线。此外，Muse Spark 预计还将取代目前驱动 WhatsApp、Instagram、Facebook 聊天机器人以及 Meta 智能眼镜系列产品的部分现有 Llama 模型。Meta 表示，仍将 API 访问范围限制在自家产品生态内，并未像热门的 OpenRouter 模型市场那样向第三方平台开放。“这项服务将运行在我们已经构建好的计算基础设施之上。”Wang 表示。

值得一提的是，此前 Meta 的 AI 战略主要强调将 Llama 系列模型开放给开源社区，而如今公司正在转向销售自主研发 AI 模型的访问权限。不过，Wang 表示 Meta 仍然“致力于开源”，并透露其 MSL 团队正在开发一个“Muse Spark 的变体版本”，未来计划开源，但他拒绝透露该版本具体发布时间。

除了 Muse Spark，Meta 前不久还发布了一款新的 AI 图像生成模型 Muse Image，其原代号为“Mango”。该模型旨在帮助 Meta 吸引创作者和广告主使用其 AI 产品。

据 Wang 最新透露，Meta 目前正在训练一个更强大的 AI 模型，代号为 Watermelon，已在关键基准上赶上 OpenAI 的 GPT-5.5，但尚未公布发布时间。
