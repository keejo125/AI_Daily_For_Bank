# 海外涨、国内守：AI收费双轨制背后的用户与利润博弈

> 原文链接：https://mp.weixin.qq.com/s/nPyinezk50cO_QMbfSo-Fw
> 公众号：财联社AI daily

文|

王耐

近日，两则消息同时在AI圈广泛传播，值得放在一起看。

一是OpenAI宣布将在几周内把Codex核心能力整合进ChatGPT，推动Codex从开发者工具向泛办公场景延展，Codex企业端计费模式也加速向Token计费转向；二是豆包宣布即将推出专业版，主要面向C端用户，计费模式为订阅制打包路线。

这是国内外两个大模型超级入口在商业化上的持续探索。目前，ChatGPT的月活已于5月破10亿大关；豆包一季度的月活用户数约3.4亿，是当前中国月活用户最多的AI应用。

这一年来，关于商业模式的探索、模型背后的算力焦虑已经被讨论了很多，大厂也在付费模式上把口子探得越来越开。

两个超级入口，商业化路径的分化或者是不同阶段的分野，指向同一个底层问题：Token是大模型处理信息的最小信息单元，消费者认可程度如何？当海外大步走向按量计费，豆包等国内厂商为何还走订阅路？

国外是怎么收费的？C 端订阅打底，B 端加速转向 Token 计费

以ChatGPT为例，面向个人用户仍然月付为主，但对高级模型的使用频次进行了限制。比如免费版中，账户每 5 小时最多可发送 10 条 GPT-5.5 消息，超出后将聊天将自动改用该模型的 mini 版本。Plus和Pro用户拥有更高的使用额度，每 3 小时最多可发送 160 条 GPT-5.5 消息，超出额度后也将临时切换至mini版本，直到Business和Pro套餐才取消对GPT-5模型的访问限制。

这与国内的大模型的订阅模式有所不同。即将推出专业版的豆包采用订阅制，设定基础版免费，以及另外三档付费档位外，并未针对消息条数有特别的限制。付费档位如下：标准版连续包月68元（年688元）、加强版200元（年2048元）、专业版500元（年5088元）。

在Kimi（月之暗面）的官方定价页面中，也未搜寻到对消息条数的限制。

Kimi的订阅价格档位

而素有“价格屠夫”之称的DeepSeek（深度求索），至今未推出任何C端付费会员套餐，仅面向开发者和小微企业的API接口。

虽然在消息使用限制上，国内外大模型在订阅制上的路径探索不尽相同。但向“更专业”方向转型、筛选付费意愿更高的重度用户，是两者的共同点。其中，个人重度用户的价值正在被重新评估。

一个数据可以说明这一点。OpenAI在6月2日的更新中披露，Codex周活跃用户数已超过500万人，其中分析师、市场人员、运营等非开发者在过去一个月的新增用户中占比达到40%。 这一数据表明，编程助手类工具正在突破开发者圈层，向泛办公场景渗透，而重度使用这些功能的个人用户，恰恰是订阅模式下最具付费意愿和留存价值的群体。

一位独立开发者Simon Willison曾于5月27日在社交媒体上发帖：“订阅了Anthropic的$100/月Max和OpenAI的$100/月Pro套餐，相当于用$200换来了价值$2,180.16的API token消耗。”

这说明，个人重度用户通过订阅享受了极大的价格杠杆。

这也引出了，订阅制面临的“用量天花板”问题。

如果个人用户月度实际 Token 消耗成本远低于订阅费用，模式可以平稳运转；一旦包月费用无法覆盖高额 Token 消耗，商业模式的可持续性就会承压。模型调用背后的算力焦虑给大厂带来的成本压力，一直都是房间里的大象。

为缓解成本压力，国外大厂开始加速探索按token用量计费的商业模式。

例如，今年4月2日，OpenAI将Codex的定价与API token用量对齐，不再按“消息条数”估算。此前，一条消息无论背后Agent读多长上下文、调用多少次工具，都只扣固定点数；新方式则按输入token、缓存token、输出token分别精准计费。这一变更适用于Plus、Pro、Business及Enterprise方案，4月23日又扩展到Enterprise计划（含教育版、医疗版、政府版等）。

OpenAI解释道，具体额度消耗取决于工作负载组合。输出量大的任务和快速模式通常比轻量任务消耗更多额度。个人用户达到使用上限后，可以购买额外积分继续使用，也可以使用API密钥运行额外任务，按标准API费率收费。

企业客户被更快地拉入按量计费轨道

除了将Codex整合进ChatGPT，OpenAI也在加速推动企业端计费向Token模式转向。在2026年4月OpenAI将Codex企业定价与API对齐，企业客户不再享受之前的巨额折扣。

据The Register报道，Anthropic已于2025年11月调整企业定价：每月20美元/席位，外加按API标准价格计费的Token用量，包月内不再附带任何免费Token配额。

业内人士认为，随着企业端向 Token 按量计费迁移的节奏持续加快，企业侧海量 Token 调用产生的增量收入，将成为未来利润增长的核心动力。

这从海外两家大模型公司的收入结构也可以一斑。

据FutureSearch推算，目前OpenAI约84%的收入来自ChatGPT付费订阅体系，其中个人Plus订阅占比55%，针对大型企业客户的ChatGPT Enterprise为OpenAI贡献了21%的收入，面向中小企业和团队用户的ChatGPT Team为OpenAI贡献了8%的收入，API接口收入占比约15%。

而一直瞄准企业和开发者市场的Anthropic，收入主要来自Claude的企业合同与付费订阅。在5月底披露，其年度经常性收入已达470亿美元，去年年底这一数字为90亿美元。

“Token计费的持续普及，正在推动AI行业估值体系迭代升级。”光大证券首席资产配置研究员王开认为，传统互联网“用户数×ARPU”的估值模型无法体现大模型算力消耗和服务层级的差异化。而大模型的成本支出、价值产出、服务频次，最终都可通过Token消耗量化，因此

行业估值正逐步向‘有效Token消耗量×单位价值’的效能型逻辑倾斜

。

“未来这类应用的核心商业价值，集中在用户活跃度、高价值场景渗透率、有效算力消耗规模以及商业化模式的迭代能力上。”

像豆包等国内应用为何坚持订阅制？

当前国内所有主流大模型面向个人用户，大部分以免费体验+包月订阅为核心模式，这与海外B端Token化、C端订阅兜底的模式形成分化。

王开认为，国内主流AI应用采用“订阅制为主、Token按量计费为辅”的混合商业化路径，是适配国内消费市场环境的选择。直接按Token收费，会让普通用户产生“用一次花一次钱”的心理负担，降低使用意愿。而且3.45亿月活的巨大用户基本盘，是豆包最大的战略资产，公司不愿意太过于激进地探索计费方式，而动摇庞大的用户基座。

此外，国内大模型端口的价格竞争依然激烈，不少APP面向个人仍然免费。面对竞争，王开认为，单一的技术壁垒已无法适配当前的行业竞争格局。随着第三方Agent入驻、平台智能体开放等生态模式普及，行业竞争从单一技术比拼，升级为技术、生态、场景、运营的综合竞争。生态构建能力已经成为AI企业的核心护城河之一。

“整体来看，短期看入口与生态红利，中期看产品与场景落地，长期依旧看技术迭代能力，三者协同构筑企业核心竞争力。”王开这样告诉记者。

以豆包为例，目前其拥有3.45亿月活，是字节AI的超级流量入口。背后的扣子（字节旗下AI编程工具）有Agent和工作流，代码助手Trae有编程工具，飞书有企业协作场景。单看每一块，字节都有东西可讲。但问题在于，字节内部不同的组织，能否真正协同起来把这件事干成。一旦建立用户粘性，迁移到其他平台的意愿则会下降。留住用户，筛选愿意付费的重度使用用户，这是字节正在下的棋。

定价分化：Token 价值因场景而异，国内外节奏分化

海外方面，大厂越来越关注模型能不能实际干活，涨价已是明牌，一代更比一代贵。

比如最新模型GPT-5.5的输入定价为5美元/百万Token，输出为30美元/百万Token，比上一代GPT-5.4贵了一倍。在OpenAI官网中，当前的定价档位已经列出一个长长的表单，档位越来越细分。

而Anthropic的定价则属于“明不变、暗涨价”。经Simon WillisonOpus 推算，Claude Opus 4.7名义单价与Opus 4.6相同，但引入了新的tokenizer（分词器），同样的文本可能被切分成约1到1.35倍的token，实际有效成本较前代约上涨40%。

国内市场则出现明显的价格分化，通用场景与生产力场景价格走向并不相同。

通用对话类大模型 API 价格持续探底。以DeepSeek为代表，5月22日，DeepSeekV4-Pro API价格在5月31日结束2.5折优惠后永久调整为原价的1/4，创下全球大模型API价格新低。5月27日，小米旗下MiMo-V2.5系列API跟进，最高降幅达到99%，不再区分上下文窗口长度。

6月3日，腾讯云宣布将DeepSeek-V4系列模型调用价格进行大幅下调，最高降幅达97.5%，成为首个大规模跟进DeepSeek降价策略的主流云厂商。据中国信通院数据，国内大模型API平均价格较2023年已累计下降超90%。

另一边，能够赋能编程、数据分析等高强度生产场景的“高端Token”价格不降反升。以编程能力见长的智谱GLM-5，在2026年2月至4月间多次上调API价格，智谱API价格在2026年第一季度同比接近翻倍。

腾讯云的AI编程助手CodeBuddy也在2026年多次提价，主要针对企业级版本。比如企业旗舰版价格由78元/人/月上调至198元/人/月，涨幅约154%；企业专享版由158元/人/月上调至316元/人/月，涨幅达100%。这也是腾讯云今年以来第三次宣布涨价。

目前，海外企业端全面转向 Token 按量计费的趋势，尚未在国内形成普遍风潮，多数本土厂商仍以订阅制或混合模式为主。但智谱GLM-5、腾讯CodeBuddy等生产力模型的涨价动作，也释放出信号：高价值场景下，Token 的定价权正在被重新定义。

王开分析，Token 经济的逐步落地，将推动 AI 产业链价值重估，也为 A 股相关赛道带来结构性机会。

“其一是高频算力调用拉动智算中心需求，算力租赁、调度、运维等业务景气度上行；二是通信运营商板块，高频的Token流转将持续激活算力网络、数据通信等基础业务。三是AI应用板块，Token计费叠加订阅制的混合商业模式，彻底改变了应用层单一的变现模式，助力企业优化营收结构与盈利模型。”

从“按人头”到“按Token”，AI行业商业化的博弈一直在进行。海外依托开发者与企业生态正在加速向Token模式转向，国内立足海量C端用户深耕订阅体系。最终哪种计价“度量衡”成为主流，又或者分属商业模式的不同阶段，不仅取决于技术和定价策略，更取决于用户对“AI服务值多少钱”这一问题的回答。
