## 事件概述

Claude Code 源代码遭"核弹级"泄漏，分析 4756 个源文件得出的 10 个秘密。

| # | 发现 | 说明 |
|---|------|------|
| 1 | **多入口架构** | 支持多种使用场景 |
| 2 | **流式响应** | 实时反馈执行状态 |
| 3 | **增量更新** | 只更新变化的部分 |
| 4 | **安全沙箱** | 代码执行在隔离环境 |
| 5 | **工具编排** | 灵活的函数调用系统 |
| 6 | **错误恢复** | 自动尝试替代方案 |
| 7 | **进度追踪** | 任务分解与状态管理 |
| 8 | **会话管理** | 跨会话恢复 |
| 9 | **插件系统** | 可扩展架构 |
| 10 | **测试覆盖** | 高测试覆盖保证稳定性 |

## 架构亮点

### 多入口架构

```
入口 1: CLI 命令行
入口 2: IDE 集成
入口 3: API 调用
入口 4: Web 界面
    ↓
统一核心引擎
```

### 安全沙箱

- 代码执行在隔离环境
- 防止恶意代码影响宿主
- 资源限制（CPU、内存、时间）

### 错误恢复机制

1. 执行失败 → 记录错误
2. 分析原因 → 生成替代方案
3. 自动重试 → 直到成功或达到上限

## 工程启示

1. **可观测性**：进度、状态、错误全程可追踪
2. **容错设计**：不是避免错误，而是优雅处理错误
3. **模块化**：核心引擎 + 插件系统，可扩展
