## 核心设计理念

### 1. 七层架构

```
入口层 → 初始化层 → 命令层 → 工具层 → 查询层 → 状态层 → UI 层
```

### 2. 命令 vs 工具

| 维度 | 命令 | 工具 |
|------|------|------|
| 触发者 | 用户 | AI 模型 |
| 前缀 | `/` | tool_use |
| 用途 | 用户意图入口 | 模型能力出口 |

### 3. QueryEngine 核心职责

- 对话生命周期管理
- 消息历史维护
- 权限拒绝追踪
- 文件状态缓存
- API 用量统计
- 流式输出接口
