## Hermes Agent

### 产品定位
Nous Research 2026年2月推出的开源Agent平台，被称为「会跟着你成长的Agent」。[1]

| 指标 | 数据 |
|------|------|
| GitHub Stars | 4万+ |
| 当前版本 | v0.8.0 |
| 更新频率 | 平均不到一周一个大版本 |
| 贡献者 | 240+ |
| 合并PR | 1400+ |

### 核心特性

| 特性 | 说明 |
|------|------|
| 自主Agent | 运行在你自己的服务器上 |
| 多平台接入 | Telegram、Discord、Slack、飞书、企业微信 |
| 内置学习闭环 | 记忆→技能→训练数据三层 |
| 定时自动化 | 自然语言cron指令 |
| Agent联邦 | 支持与OpenClaw互操作 |

### 三层学习闭环

| 层级 | 说明 | 价值 |
|------|------|------|
| 记忆 | MEMORY.md + USER.md + FTS5检索 | 跨会话上下文 |
| 技能 | 从经验中自动创建skill | 复用积累 |
| 训练数据 | 工具调用记录用于模型训练 | 越用越强 |

**核心公式**：
```
记忆沉淀 → 自动提炼技能 → 生成训练数据 → 模型能力提升 → 更强Agent
```

### 版本演进路线

| 版本 | 日期 | 主题 | 重点 |
|------|------|------|------|
| v0.5.0 | 2026-03-28 | hardening | 50+安全修复 |
| v0.7.0 | 2026-04-03 | resilience | 可插拔记忆、凭证轮换 |
| v0.8.0 | 最新 | intelligence | 后台任务通知、模型切换 |

**核心判断**：长期运行才是Agent真正的工程挑战（不是"不够聪明"，而是"跑着跑着崩了"）

## 与我行工具对比

### Hermes vs Qoder/Claude Code

| 维度 | Hermes Agent | Claude Code | Qoder |
|------|---------------|-------------|-------|
| 部署方式 | 自托管（5$/月VPS） | 客户端工具 | 本地部署 |

| 银行适用性 | ⚠️ 需安全评估 | ✅ | ✅ |

## 关键启示

### 对我行的价值

1. **私有化部署**：数据留在用户手里，符合银行安全合规
2. **技能沉淀机制**：解决"人员流动知识流失"问题
3. **训练数据闭环**：使用数据可用于模型训练
4. **多平台接入**：支持飞书、企业微信

### 风险点

1. **安全评估**：开源组件需要安全扫描
2. **运维成本**：7x24运行需要专业运维
3. **厂商锁定**： Nous Research融资背景需评估
