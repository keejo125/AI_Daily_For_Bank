# 女娲 Skill：知识蒸馏新范式

> 来源：微信公众号  
> 收录：2026-04-06

## 什么是女娲 Skill

开源技能包，可以将任何人的知识和能力"蒸馏"成 AI 可复用的 Skill。

**核心理念**：
- 把个人/团队的领域知识封装成 Skill
- AI 可以直接调用，无需从零学习
- 知识可以"蒸馏"和"传承"

## 应用场景

| 场景 | 说明 |
|------|------|
| **离职同事知识传承** | 把离职同事的经验、代码风格蒸馏成 Skill |
| **团队规范沉淀** | 代码规范、业务规则封装成 Skill |
| **个人能力复用** | 自己的领域知识变成可复用的工具 |

## 热点事件

「同事.Skill」冲上热搜——离职同事已被"炼化"！

**争议点**：
- 知识蒸馏的边界在哪里？
- 未经同意能否蒸馏他人知识？
- 隐私和知识产权问题

## 技术启示

1. **知识可产品化**：领域知识可以封装成可交易、可复用的产品
2. **团队知识资产**：不再是"人走知识丢"，而是可沉淀、可传承
3. **个人 IP 新形态**：知识可以被打包成 Skill 分发

## 与 Karpathy 方法论的关联

| Karpathy 方法 | 女娲 Skill |
|--------------|-----------|
| Raw 层 | 原始资料、对话记录 |
| Wiki 层 | 结构化的知识 Skill |
| AI 编译 | 自动蒸馏成 Skill |

**本质**：女娲 Skill 是 Karpathy "AI 编译知识"理念的产品化实现。
